## 摘要
视觉-语言-动作模型在机器人操作中取得了显著进展，但仍存在一个关键局限：缺乏三维场景理解。这一不足表现为三个相互交织的挑战：在未强制多视角一致性的情况下对三维空间位置的提取能力较弱、三维实例理解不足，以及在遮挡情况下推理脆弱。尽管已有成熟的三维感知方法，但由于架构不兼容以及对高成本实例级标注的强依赖，将其直接集成到 VLA 流水线中仍受到阻碍。为应对上述挑战，我们提出 3DVLA，这是一个即插即用框架，可在无需额外人工标签且不舍弃 VLM 先验的情况下，将鲁棒的三维推理注入预训练 VLA。具体而言，3DVLA 通过以下方式解决这三个挑战：（1）在所有模态中采用显式多视角一致性约束的广泛三维特征编码，以及一种空间条件化几何聚合方法；（2）引入带有高级实例 token 的实例估计模块，以实现三维实例感知；（3）构建一个掩码自监督三维编码分支，并保留其预测器用于视觉 token 补全，以处理遮挡。我们将 3DVLA 集成到多个 VLA 基线中，并在 LIBERO-Plus 和 RoboTwin 2.0 上进行评估。结果显示，操作性能获得了一致且显著的提升，验证了我们方法的有效性及其即插即用兼容性。

## Abstract
Vision-Language-Action models have achieved remarkable progress in robotic manipulation, yet they suffer from a critical limitation: a lack of 3D scene understanding. This deficiency manifests as three intertwined challenges: weak extraction of 3D spatial positions without enforcing multi-view consistency, inadequate 3D instance understanding, and fragile reasoning under occlusion. Although mature 3D perception methods exist, their direct integration into VLA pipelines is hindered by architectural incompatibility and by heavy reliance on costly instance-level annotations. To address the above challenges, we propose 3DVLA, a plug-and-play framework that injects robust 3D reasoning into pretrained VLAs without requiring extra manual labels or discarding VLM priors. Specifically, 3DVLA tackles the three challenges through: (1) pervasive 3D feature encoding with explicit multi-view consistency constraints across all modalities and a Spatially-Conditioned Geometry Aggregation method, (2) an instance estimation module with high-level instance tokens for 3D instance awareness, and (3) a masked self-supervised 3D encoding branch that retains its predictor for visual token completion to handle occlusions. We integrate 3DVLA with multiple VLA baselines and evaluate on LIBERO-Plus and RoboTwin 2.0. Results show consistent and significant gains in manipulation performance, validating both the effectiveness and plug-and-play compatibility of our approach.

## 1. 核心问题与整体含义：研究动机与背景

本文关注的是当前视觉-语言-动作模型（Vision-Language-Action Models, VLAs）在机器人操作任务中的一个关键短板：**缺乏可靠的三维场景理解能力**。

现有 VLA 模型通常基于 2D 图像、语言指令和动作序列进行建模，虽然借助大规模视觉语言预训练模型获得了较强的语义理解和泛化能力，但在真实机器人操作场景中仍容易失败。原因在于机器人执行动作本质上依赖三维空间信息，例如物体的 3D 位置、朝向、遮挡关系、实例边界以及与机械臂末端执行器之间的相对几何关系。

论文将这一问题具体拆解为三个相互关联的挑战：

- **三维空间位置提取能力弱**
  - 现有 VLA 多数只处理单视角或简单多视角 2D 特征。
  - 缺少显式的 3D 坐标建模和多视角一致性约束。
  - 因此模型难以在统一的度量空间中理解物体位置和 6-DoF 关系。

- **三维实例理解不足**
  - 标准 VLA 往往把物体当作图像中的 2D 区域。
  - 缺少对物体体积、3D 边界、实例身份和几何结构的建模。
  - 在需要抓取特定部件、放置到某个空间位置等任务中容易失败。

- **遮挡场景下推理脆弱**
  - 当目标物体被部分遮挡或相机视角变化时，2D 模型只能看到不完整视觉信息。
  - 缺少对不可见结构的补全能力，导致动作决策不稳定。

已有 3D 感知方法虽然能提供三维理解，但直接用于 VLA 存在两个障碍：

- 架构上与基于 VLM 的 VLA 模型不兼容，可能破坏已有视觉语言预训练先验。
- 很多 3D 感知方法依赖昂贵的实例级或语义级人工标注，不适合大规模机器人数据。

因此，本文提出 **3DVLA**：一个即插即用的 3D 增强框架，用于在不额外引入人工实例标注、不放弃 VLM 预训练能力的前提下，为已有 VLA 注入三维空间、实例和遮挡补全能力。

## 2. 方法论：核心思想、关键技术与流程

3DVLA 的总体思想是：  
**将多视角 RGB-D 视觉特征提升到统一 3D 空间中，构建具有多视角一致性的 3D 记忆；再从中提取 3D 实例 token，并通过自监督预测器补全被遮挡几何；最后将这些空间条件化的几何 token 输入原有 VLA 的动作专家。**

整体框架包含四个核心模块：

1. **Multi-View Spatial Fusion：多视角空间融合**
2. **Object-Centric 3D Instance Module：以对象为中心的 3D 实例模块**
3. **Coordinate-Driven 3D Self-Supervised Predictor：坐标驱动的 3D 自监督预测器**
4. **Spatially-Conditioned Geometry Aggregation：空间条件化几何聚合**

### 2.1 多视角空间融合：构建统一 3D 记忆

输入为多视角特征图：

- 来自多个相机视角的视觉特征：
  - \(X_{fmap} \in \mathbb{R}^{V \times C \times H \times W}\)
- 其中：
  - \(V\)：视角数量
  - \(C\)：通道数
  - \(H, W\)：特征图尺寸

论文不采用昂贵的体素化，而是将特征图展平为 token 序列：

- 得到 \(N = V \times H \times W\) 个视觉 token。
- 每个 token 关联一个由深度图和相机参数反投影得到的 3D 坐标。

核心做法：

- 使用一个坐标投影网络将 3D 坐标编码为 3D 位置嵌入。
- 将视觉特征与 3D 位置嵌入相加。
- 输入 Transformer Encoder，使不同视角中对应同一物理空间位置的特征能够相互对齐。

此外，论文在注意力的 Query 和 Key 上引入 **连续 3D RoPE（Rotary Position Embedding）**：

- 分别对 \(x, y, z\) 三个轴进行连续位置编码。
- 用于向注意力机制注入欧氏相对距离先验。
- 这样既保留绝对 3D 坐标信息，又能建模局部相对几何关系。

该模块的作用是：

- 将多视角 2D 特征转换为全局一致的 3D memory。
- 为后续实例提取和遮挡补全提供空间基础。

### 2.2 以对象为中心的 3D 实例模块

该模块的目标是从全局 3D memory 中提取物体级表示，而不是依赖 2D mask 或 2D box。

#### 3D Probe 初始化

模型初始化一组 3D 状态探针：

- 每个 probe 包含：
  - 语义状态 \(c_j\)
  - 3D 参考点 \(p_j\)

这些 probe 直接存在于连续三维空间中，用于寻找和表示场景中的物理实体。

#### 多尺度特征采样

全局 memory 被解码为多尺度语义金字塔，包括 stride 8、16、32 的特征。

对每个 3D probe：

1. 将其 3D 点 \(p_j\) 投影到每个相机视角。
2. 在对应 2D 特征图上进行 deformable cross-attention 采样。
3. 采集多尺度、多视角视觉证据。
4. 聚合为该 3D 实例的语义表示。

为了避免 3D 点投影时深度接近零带来的数值不稳定，论文引入了深度 clamp：

- 对投影深度 \(Z_c\) 使用下界 \(\epsilon = 0.01\)。

#### 视角相关预测与多视角一致性

同一个 3D 物体在不同相机视角中可能呈现不同的 2D 形状。因此论文没有强制一个 probe 直接预测所有视角相同的 2D mask，而是：

- 用共享的 3D 语义状态表示物体身份。
- 用视角相关的位置编码调节 2D box 和 mask 的预测。

这样可以兼顾：

- 3D 实例身份的一致性。
- 2D 投影形状的视角差异。

#### 遮挡感知 gating

不同视角中某个物体可能被遮挡或不可见。论文设计了自适应 gating：

- 对每个视角计算有效性权重。
- 遮挡、不清晰或视野外的视角会被自动降低权重。
- 清晰视角的信息会更多地更新该实例 probe。

#### 有界坐标细化

probe 的 3D 位置需要迭代修正。论文使用有界坐标更新：

- 使用 MLP 预测坐标偏移。
- 用 `tanh` 限制每层最大偏移量。
- 再乘以物理尺度因子 \(\alpha\)。

这样可以防止训练过程中 3D 点位置发散。

### 2.3 全局 3D 匹配与联合优化

为了训练 3D 实例模块，论文没有使用额外人工 3D 实例标注，而是借助冻结的开放世界感知模型 **VL-SAMv2** 生成伪标签：

- 生成 2D bounding boxes。
- 生成 2D masks。
- 结合深度图将 2D 中心反投影到 3D，得到伪 3D 中心。

匹配方式：

- 不在各视角分别做 2D 匹配。
- 而是在全局 3D 空间中进行 Hungarian matching。
- 匹配代价综合：
  - 分类代价
  - 2D box 代价
  - mask 代价
  - 3D 距离代价

联合损失包括 6 项：

- 分类 focal loss
- 2D box L1 loss
- 2D GIoU loss
- mask cross-entropy loss
- mask DICE loss
- 3D 坐标 L1 loss

这一设计的关键意义是：

- 避免不同视角下同一物体 ID 不一致。
- 将 2D 视觉区域稳定锚定到统一 3D 物理空间。

### 2.4 坐标驱动的 3D 自监督预测器：用于遮挡补全

本文的一个重要创新点是重新利用 masked self-supervised learning 中通常被丢弃的 predictor。

传统 masked autoencoding 或 JEPA 类方法在预训练后往往只保留 encoder，丢弃 predictor。本文认为：

- 如果引入统一 3D 位置编码，predictor 可以在推理时用于查询不可见位置的视觉 token。
- 因此它可以承担遮挡几何补全功能。

#### 训练阶段

论文定义两类坐标：

- **Mask Coordinates**
  - 对可见区域进行 mask，用于自监督训练。
- **Completion Coordinates**
  - 对不可见几何区域采样，用于学习补全。

训练时使用非对称多视角 masking：

- 随机遮挡一部分视角或区域。
- 强迫模型利用其他视角和 3D 几何关系预测被遮挡 token。
- 避免模型只依赖局部 2D 插值。

#### 补全坐标生成

为了得到需要补全的位置，论文流程如下：

1. 将 2D mask 反投影为粗糙点云。
2. 使用动态密度聚类去噪。
3. 使用预训练点云补全网络 PointAttn 预测完整形状。
4. 采样远离已观测点的区域，得到不可见位置。
5. 使用 farthest point sampling 生成均匀的 completion coordinates。

#### 混合注意力机制

预测器同时使用：

- 连续 3D Fourier 位置编码
- 离散相机视角 embedding
- 连续 3D RoPE

其注意力结构分为：

- cross-attention：
  - target query 从可见上下文中提取信息。
- self-attention：
  - target token 之间交换空间结构信息。

最终输出：

- 被 mask 位置的预测 token。
- 被遮挡补全位置的 completion token。

#### 自监督损失

为了避免连续特征预测塌缩为平均向量，论文设计三项损失：

- Smooth-L1 reconstruction loss：
  - 对齐特征幅值。
- Cosine loss：
  - 对齐特征方向。
- Variance regularization：
  - 防止所有位置输出过于相似。

教师网络使用 EMA 更新，提供稳定蒸馏目标。

### 2.5 空间条件化几何聚合

最后，3DVLA 需要将 3D 实例 token 和补全 token 接入下游 VLA 动作专家，同时保持即插即用。

#### 末端执行器中心的相对位置编码

机器人动作本质上是相对于夹爪或末端执行器执行的。因此论文不只使用绝对世界坐标，而是计算：

- \(\Delta p = p_{obj} - p_{ee}\)

即物体或补全点相对于机械臂末端执行器的位置。

然后使用多频正余弦函数编码该相对位置，并投影到 token 维度。

这样可以直接向动作专家提供更可操作的空间信息。

#### 不确定性引导的实例几何路由

直接把补全几何特征加入实例 token 可能会污染原本清晰、可靠的视觉表示。为此论文设计 uncertainty-guided routing：

1. 对每个实例，找到其附近的 K 个 completion points。
2. 使用 cross-attention 聚合局部几何上下文。
3. 注意力同时考虑：
   - 语义相似度
   - 3D 物理距离偏置
4. 根据实例分类 logit 估计不确定性：
   - 置信度高，则少吸收补全几何。
   - 置信度低或遮挡严重，则更多吸收补全几何。

路由 gate 初始接近 0，保证训练初期近似恒等映射，避免破坏预训练特征。

## 5. 实验数量与充分性分析

### 实验数量概览

论文实验主要包括以下几类：

#### 主实验

- LIBERO-Plus：
  - 7 类扰动。
  - 与 9 个左右 VLA 或机器人策略方法对比。
- RoboTwin 2.0：
  - Easy / Hard 两种设置。
  - 与 DP、ACT、RDT、DP3、π0、X-VLA 等方法对比。
  - 在 π0 和 X-VLA 两个不同 backbone 上验证即插即用性。

#### 消融实验

在 RoboTwin 2.0 和 LIBERO-Plus 上进行了模块级消融：

- baseline
- 加入 3D Instance Module
- 加入 3D Self-Supervised Predictor
- 加入 Uncertainty-Guided Routing

还包括一些微观消融：

- 去掉 Spatial Fusion。
- 用 2D Matching 替代 Global 3D Matching。
- 去掉 End-Effector Relative Positional Encoding。
- 比较 25% masking 和 50% masking。

### 是否充分

总体上，实验设计较为充分，理由包括：

- 覆盖了两个性质不同的机器人 benchmark：
  - 一个侧重扰动鲁棒性。
  - 一个侧重物理交互和 3D 操作。
- 不只在单一 baseline 上验证，而是在 π0、π0.5、X-VLA 等模型上测试。
- 既有主结果，也有逐模块消融，能够说明各组件贡献。
- 对遮挡、视角变化、多视角一致性等核心问题有针对性评估。

### 公平性与客观性

相对公平之处：

- 使用官方训练和评估协议。
- 使用 success rate 作为通用指标。
- 与较多已有 VLA 和机器人策略方法对比。
- 强调 VLM backbone 冻结，主要训练新增 3D 模块与动作专家，体现插件性质。

需要注意的地方：

- 实验均为仿真 benchmark，未报告真实机器人实验。
- 部分伪标签依赖外部感知模型 VL-SAMv2，性能可能受到该模型质量影响。
- Completion coordinates 依赖预训练点云补全网络 PointAttn，也引入了额外先验。
- 对不同相机数量、深度噪声、真实传感器误差的敏感性没有系统展开。

## 6. 主要结论与发现

论文的主要结论包括：

- **3D 空间建模能显著增强 VLA 的操作鲁棒性**
  - 通过将多视角特征锚定到统一 3D 坐标系，模型能更稳定地处理视角变化和空间扰动。

- **3D 实例 token 有助于提升对象级操作能力**
  - 相比纯 2D 区域建模，3D 实例模块能更好地表示物体身份、位置和跨视角一致性。

- **遮挡补全对困难任务尤其重要**
  - 在 RoboTwin 2.0 Hard 设置中，加入自监督 3D predictor 带来明显提升。
  - 说明补全不可见结构能改善复杂操作中的动作决策。

- **不确定性引导融合比直接拼接更有效**
  - 只有在实例检测不确定、遮挡严重时才加强补全几何输入，可以避免污染高置信视觉特征。

- **3DVLA 具有较好的即插即用性**
  - 能接入不同 VLA backbone，例如 π0、π0.5、X-VLA。
  - 不需要额外人工实例标注。
  - 保留 VLM 预训练先验。

## 7. 优点与亮点

### 方法设计亮点

- **兼容现有 VLA/VLM 架构**
  - 没有重建整个 VLA 系统，而是作为 3D 增强模块插入。
  - 降低了迁移到已有模型的难度。

- **不依赖额外人工实例标注**
  - 使用冻结开放世界感知模型产生伪标签。
  - 避免高成本 3D instance annotation。

- **多视角一致性建模清晰**
  - 通过 3D 坐标嵌入、Transformer 融合和 3D RoPE，使多视角特征在统一空间中对齐。

- **3D 实例建模较完整**
  - 包含 3D probe、可变形采样、视角 gating、坐标 refinement、global 3D matching 等设计。
  - 比简单拼接点云或深度图更系统。

- **遮挡补全思路有创新性**
  - 将 masked self-supervised predictor 保留下来用于推理阶段的 token completion。
  - 将自监督表征学习与机器人遮挡推理结合起来。

- **末端执行器中心编码符合机器人控制逻辑**
  - 使用物体相对 gripper 的位置，而非仅用世界坐标。
  - 更贴近动作生成需求。

- **实验覆盖多个 backbone**
  - 展示了方法不是只对单一模型有效。

## 8. 不足与局限

### 实验覆盖限制

- **缺少真实机器人验证**
  - 所有实验均在仿真 benchmark 上完成。
  - 真实世界中存在传感器噪声、深度缺失、相机标定误差、光照反射、动态遮挡等问题，方法表现尚未验证。

- **benchmark 数量仍有限**
  - 只使用 LIBERO-Plus 和 RoboTwin 2.0。
  - 未覆盖更多真实机器人数据集或开放环境任务。

- **长时序记忆能力不明确**
  - 方法强调当前多视角观测和遮挡补全，但对长期记忆、历史状态融合讨论较少。
  - 对长时序任务中的累积误差和状态漂移缺乏分析。

### 方法依赖与偏差风险

- **依赖深度图和相机参数**
  - 多视角 3D 融合需要可靠 RGB-D、内参、外参。
  - 如果深度质量差或标定不准，3D memory 可能出现错误。

- **依赖外部伪标签模型**
  - 3D 实例模块训练依赖 VL-SAMv2 生成 2D box/mask。
  - 如果 VL-SAMv2 漏检、误分割或类别偏置，可能影响实例 token 质量。

- **依赖预训练点云补全网络**
  - Completion coordinates 的生成借助 PointAttn。
  - 该网络的形状先验可能对新类别、非刚体或复杂工具不稳定。

- **伪 3D 中心可能不精确**
  - 由 2D mask 中心结合深度反投影得到的 3D centroid 可能不是物体真实几何中心。
  - 对透明、反光、细长或遮挡严重物体可能存在误差。

### 计算与部署限制

- **虽然参数增加不大，但系统复杂度较高**
  - 包含多视角融合、实例解码、自监督预测、几何路由等多个模块。
  - 工程实现复杂度高于普通 VLA。

- **实时性未充分报告**
  - 论文称推理开销较小，但没有详细给出每步动作推理延迟、FPS 或端到端实时性能。
  - 对真实机器人在线控制部署仍需进一步验证。

- **最多实例数和补全点数受限**
  - 文中设置最多 16 个实例 token、最多 6 个遮挡实例、每个实例 5 个 completion query。
  - 在更复杂、拥挤、多物体场景下是否足够仍不确定。

### 评价指标相对单一

- 主要使用任务 success rate。
- 缺少对以下指标的系统分析：
  - 3D 实例检测精度。
  - 遮挡补全质量。
  - 多视角一致性误差。
  - 动作轨迹平滑性。
  - 失败类型分布。
  - 推理延迟和资源占用。

## 总体评价

3DVLA 是一篇针对 VLA 三维理解不足问题的系统性工作。它不是简单地把点云输入 VLA，而是围绕机器人操作需求构建了较完整的 3D 增强机制，包括多视角一致空间融合、3D 实例 token、遮挡几何补全以及末端执行器中心的空间条件聚合。实验表明，该方法能稳定提升多个 VLA baseline 在 LIBERO-Plus 和 RoboTwin 2.0 上的成功率，尤其对视角变化、遮挡和复杂几何交互场景有帮助。

不过，该方法目前主要在仿真环境中验证，且依赖深度图、相机标定、伪标签感知模型和点云补全网络。其真实机器人部署能力、实时性、对传感器误差的鲁棒性，以及在更复杂开放世界场景中的泛化能力仍需要进一步研究。
