## 摘要
我们提出 SOMA，即用于视觉-语言-动作（VLA）模型中视野外操作的空间记忆框架。现有大多数 VLA 隐含地假设与任务相关的物体始终可见，导致当目标超出相机视场时，模型表现出脆弱且反应式的行为。SOMA 通过为 VLA 配备由可移动头部相机获取的多视角观测所构建的持久空间记忆，解决了这一局限，使其能够在当前视觉视锥之外进行推理。该框架由三个组件组成：空间记忆构建，通过扫描将按角度组织的观测聚合为统一的空间-语义表征；动态记忆精炼，用于随时间维持全局一致性；以及上下文记忆检索，在操作过程中激活与指令相关的空间线索。我们在五项具有挑战性的真实世界视野外操作任务上评估了 SOMA，包括目标物体初始不可见的多步骤和双臂场景。实验结果表明，SOMA 不仅提高了任务成功率，还诱导出质性不同的操作行为，表现为更快的目标定位、更少的视角搜索，以及在部分可观测条件下接近一次成功的抓取。RoboCasa GR1 和 SimplerEnv 上的附加实验进一步验证了 SOMA 的记忆设计在传统完全可观测设置下的有效性。代码将很快发布。

## Abstract
We introduce SOMA, the Spatial Memory framework for Out-of-Vision Manipulation in Vision-Language-Action (VLA) models. Most existing VLAs implicitly assume that task-relevant objects are always visible, leading to brittle and reactive behaviors when targets fall outside the camera's field of view. SOMA addresses this limitation by equipping VLAs with a persistent spatial memory constructed from multi-view observations acquired via a movable head camera, enabling reasoning beyond the current visual frustum. The framework consists of three components: Spatial Memory Construction, which aggregates angular-wise observations into a unified spatial-semantic representation through scanning; Dynamic Memory Refinement, which maintains global consistency over time; and Contextual Memory Retrieval, which activates instruction-relevant spatial cues during manipulation. We evaluate SOMA on five challenging real-world out-of-vision manipulation tasks, including multi-step and dual-arm scenarios where target objects are initially invisible. Experimental results show that SOMA not only improves task success rates, but also induces qualitatively different manipulation behaviors, with faster target localization, reduced viewpoint search, and near one-shot grasping under partial observability. Additional experiments on RoboCasa GR1 and SimplerEnv further validate the effectiveness of SOMA's memory design under conventional fully observable settings. Code will be released soon.

## 1. 核心问题与整体含义：研究动机与背景

本文关注的是 **Vision-Language-Action（VLA）机器人操作模型在“视野外操作”中的失效问题**。

现有多数 VLA 模型通常依赖当前相机观测进行动作预测，隐含假设是：

- 任务相关物体始终在当前视野中；
- 机器人只需要根据即时视觉输入、语言指令和自身状态生成动作；
- 不需要长期保存、更新或检索场景中的空间信息。

但在真实机器人操作中，这一假设往往不成立。例如：

- 目标物体可能在当前头部相机或腕部相机之外；
- 放置目标位置可能被遮挡或不在视野中；
- 多阶段任务中，机器人需要记住之前看到过的物体或容器位置；
- 双臂协作时，不同手臂与头部相机之间存在视角不一致问题。

论文将这种问题称为 **Out-of-Vision（OOV）Manipulation，即视野外操作**。  
当目标不在当前视觉视锥中时，纯反应式 VLA 模型无法定位目标，容易出现：

- 搜索时间长；
- 头部视角反复调整；
- 抓取姿态不稳定；
- 多阶段任务中遗忘目标位置；
- 双臂协调失败。

因此，论文的核心主张是：

> 仅依靠当前视野和模型内部的空间推理不足以完成可靠的视野外操作，机器人需要一种持久的、可更新的、可检索的空间语义记忆。

为此，作者提出 **SOMA（Spatial Memory for Out-of-Vision Manipulation in VLA）**，通过可动头部相机主动扫描场景，构建全局空间语义记忆，并在执行过程中动态更新与检索，从而让 VLA 模型能够在当前视野之外进行操作决策。

## 2. 方法论：SOMA 的核心思想与技术流程

SOMA 的核心思想是将 VLA 从“当前视角驱动的反应式策略”转变为“空间记忆驱动的操作策略”。

整体框架包含三个关键模块：

1. **Spatial Memory Construction：空间记忆构建**
2. **Dynamic Memory Refinement：动态记忆精炼**
3. **Contextual Memory Retrieval：上下文记忆检索**

### 2.1 空间记忆构建：从多视角扫描构建全局场景记忆

在操作开始前，如果检测模块发现语言指令中的目标物体不在当前视野中，机器人会启动头部相机扫描。

扫描过程得到一段多视角视频序列：

\[
V = \{f_i\}_{i=1}^{N_v}
\]

系统从中均匀采样若干帧，并对每一帧执行以下处理：

- **VGGT**：估计相机位姿与粗略 3D 几何；
- **YOLO / YOLO-World**：检测物体类别与 2D bounding box；
- **DINOv3**：提取物体区域的高维视觉语义特征。

对于每个检测到的物体实例：

- 从 DINOv3 特征图中裁剪目标区域；
- 平均池化得到实例级外观特征；
- 使用 VGGT 的几何估计将 2D 检测框提升到全局 3D 坐标系；
- 得到该物体的 3D bounding box。

随后，系统进行跨视角实例融合：

- 同类别物体之间比较 DINOv3 外观相似度；
- 同时比较 3D bounding box 的空间一致性；
- 相似度超过阈值的实例被认为是同一物体并进行合并；
- 对外观特征与 3D 几何进行平均聚合。

最终，每个全局物体实例表示为三元组：

\[
(f_k, c_k, b_k)
\]

其中：

- \(f_k\)：物体外观语义特征；
- \(c_k\)：物体类别；
- \(b_k\)：3D bounding box。

然后，3D bounding box 会经过可学习的位置编码模块：

\[
p_k = \Phi_{pos}(b_k)
\]

物体外观特征经过记忆投影模块：

\[
\Phi_{mem}(f_k)
\]

两者相加形成最终记忆 token：

\[
m_k^0 = \Phi_{mem}(f_k) + p_k
\]

所有物体 token 组成初始全局场景记忆：

\[
M_0 = \{m_k^0\}_{k=1}^{N_I}
\]

这个记忆同时包含：

- 物体类别与外观语义；
- 物体在全局坐标中的空间位置；
- 多视角融合后的稳定对象表示。

---

### 2.2 动态记忆精炼：执行过程中持续更新空间记忆

在机器人操作过程中，场景可能发生变化：

- 物体被拿起或移动；
- 目标被遮挡；
- 新物体进入视野；
- 多阶段任务导致场景状态更新。

因此，SOMA 不只使用初始记忆 \(M_0\)，还会根据当前头部相机观测动态维护记忆。

当前时刻 \(t\)，系统从头部相机观测中提取当前记忆 token：

\[
M_t = \{m_j^t\}_{j=1}^{N_t}
\]

然后将当前观测中的实例与历史记忆中同类别实例进行匹配。匹配成功后，使用相似度感知的融合机制更新记忆。

论文定义两个分数：

1. **语义相似度分数**：

\[
s_{kj}^t = \sigma(\Phi_{sim}([m_k^{t-1} - m_j^t]))
\]

2. **动态融合分数**：

\[
g_{kj}^t = \sigma(\Phi_{fuse}([m_k^{t-1}, m_j^t]))
\]

二者相乘得到自适应更新系数：

\[
\alpha_{kj}^t = g_{kj}^t \cdot s_{kj}^t
\]

然后使用类似 EMA 的方式更新记忆：

\[
m_k^t = \alpha_{kj}^t m_j^t + (1-\alpha_{kj}^t)m_k^{t-1}
\]

该设计的作用是：

- 当新观测与旧记忆一致时，平滑更新；
- 当场景发生真实变化时，较快吸收新信息；
- 未匹配到的新实例会加入记忆；
- 暂时未观测到的历史实例会保留，以应对遮挡和视野外情况。

更新后的记忆记为：

\[
\hat{M}_t
\]

### 2.3 上下文记忆检索：根据语言指令激活相关空间线索

SOMA 将空间记忆注入 VLA 的视觉语言表示中。

给定 VLM 产生的视觉-语言 token：

\[
X_{vl} = \{x_i\}_{i=1}^{N_q}
\]

系统将其作为 query，将更新后的空间记忆 \(\hat{M}_t\) 作为 key-value 记忆库。

首先将记忆 token 投影到 VLM 特征空间：

\[
\tilde{m}_k^t = \Phi_{align}(\hat{m}_k^t)
\]

然后通过 cross-attention 进行检索：

\[
X_{boost} = \text{softmax}\left(\frac{QK^\top}{\sqrt{C}}\right)V
\]

其中：

- \(Q\)：视觉语言 token；
- \(K,V\)：空间记忆 token；
- \(X_{boost}\)：注入空间记忆后的增强表示。

这些增强 token 会输入到 DiT blocks 与动作解码器中，最终生成动作 chunk：

\[
\{\hat{a}_t, \hat{a}_{t+1}, ..., \hat{a}_{t+H-1}\}
\]

因此，SOMA 的完整流程可以概括为：

1. 判断目标是否在当前视野中；
2. 若不可见，则头部相机主动扫描；
3. 从多视角观测构建全局空间语义记忆；
4. 执行过程中持续更新记忆；
5. 根据语言指令检索相关空间 token；
6. 将记忆增强后的表示输入 VLA 动作解码器；
7. 输出机器人动作序列。

- **GR00T-N1.5**；
- **SOMA**。

此外，作者还做了真实世界消融：

- **Scan+GR00T**：扫描后用检测到的目标帧作为固定输入，无持久记忆；
- **No-Scan SOMA**：不进行多视角扫描，只用第一帧初始化记忆；

- **Full SOMA**：完整模型。

## 4. 资源与算力

论文明确给出了部分训练与推理资源。

### 4.2 推理算力

真实世界推理运行在：

- 一台配备 **NVIDIA RTX 4090 GPU** 的服务器上。

机器人本体配备：

- **NVIDIA Jetson AGX Orin 64GB**，但文中主要推理似乎在外部 4090 服务器上执行。

### 4.3 预处理资源与延迟

空间记忆构建中的感知预处理使用：

- VGGT；
- YOLO；
- DINOv3。

论文报告了不同模块的时间和显存开销，并指出：

- StarVLA：约 1.26 秒 / chunk；
- SpatialVLA：约 1.45 秒 / chunk；
- GR00T-N1.5：约 1.30 秒 / chunk；
- SOMA：约 1.58 秒 / chunk。

因此，SOMA 增加了一定延迟，但仍在可接受范围内。

## 6. 主要结论与发现

论文的主要结论如下。

### 6.1 空间记忆显著提升视野外操作能力

在五个真实世界 OOV 任务中，SOMA 均取得最高成功率。尤其在复杂任务中，例如：

- Sequential Dual-Object PnP；
- Dual-Arm Coordination PnP；

SOMA 与其他方法的差距更明显，说明持久空间记忆对多阶段和双臂操作非常关键。

### 6.2 SOMA 改变了机器人行为模式

相比 GR00T-N1.5，SOMA 不只是成功率更高，还表现出更有效率的操作行为：

- 更快首次锁定目标；
- 更短头部搜索路径；
- 更少视角修正；
- 更少抓取尝试；
- 更短 Time-to-Grasp。

论文报告多个指标上 SOMA 相比 GR00T-N1.5 有约 40%–60% 的改善。

这说明 SOMA 不是简单提升了一点识别性能，而是让机器人从“盲目搜索”变成“基于记忆定位”。

### 6.3 扫描本身不够，持久记忆才是关键

消融实验显示：

- Scan+GR00T 表现较弱；
- No-Scan SOMA 略优于 Scan+GR00T；
- Scan-only SOMA 进一步提升；
- Full SOMA 最好。

这说明：

> 多视角扫描可以提供信息覆盖，但真正带来稳定提升的是持久空间记忆以及执行过程中的动态更新与检索。

### 6.4 在完全可观测仿真任务中，空间记忆同样有益

在 RoboCasa GR1 和 SimplerEnv 中，即使任务相关对象通常可见，SOMA 仍能提升表现。

这说明空间记忆不仅用于解决 OOV 问题，也能为一般机器人操作提供更强的空间结构归纳偏置。

### 6.5 几何、语义和动态更新都不可或缺

消融表明：

- 去掉几何线索会降低成功率；
- 去掉物体语义会降低成功率；
- 去掉动态更新下降最大。

因此，SOMA 的有效性来自三者结合：

- 3D 几何定位；
- 物体级语义识别；
- 时序一致性维护。

## 7. 优点与亮点

### 7.1 问题定义清晰且具有现实意义

论文明确指出现有 VLA 的一个关键盲点：  
大多数模型只在当前视野内做决策，缺乏对“看过但当前不可见”物体的持久记忆。

这一问题在家庭服务、工业整理、双臂操作和长时程任务中非常常见。

### 7.2 方法设计模块化，易于理解和扩展

SOMA 被拆分为三个清晰模块：

- 构建；
- 更新；
- 检索。

这种设计便于替换具体感知模块，例如未来可将 YOLO、DINOv3、VGGT 替换成更强的检测、语义或 3D 重建模型。

### 7.3 将主动感知与 VLA 决策结合

论文不是单纯让机器人“多看几眼”，而是将头部扫描得到的信息转化为可供 VLA 检索的空间记忆。

这比传统主动视角搜索更进一步。

### 7.4 真实世界实验有说服力

作者搭建了真实机器人平台，并采集了大量遥操作示范，实验覆盖：

- 不可见目标；
- 不可见放置位置；
- 多阶段任务；
- 双臂协作。

这使得论文不只是仿真验证，实际应用意义更强。

### 7.5 行为指标分析较细致

论文不只报告 success rate，还分析：

- 搜索路径长度；
- 首次注视时间；
- 视角修正次数；
- 抓取尝试次数；
- 抓取耗时。

这些指标更好地揭示了 SOMA 的行为机制。

### 7.6 消融实验较全面

论文对以下因素均进行了分析：

- 是否扫描；
- 是否构建记忆；
- 是否动态更新；
- 是否使用几何；
- 是否使用语义；
- 检索模块深度；
- 采样间隔；
- 几何来源；
- 推理延迟。

这增强了结论可信度。

## 8. 不足与局限

### 8.1 真实世界 benchmark 仍较小且自定义

虽然真实实验很有价值，但 benchmark 是作者自建的，主要集中在桌面 Pick-and-Place 场景。

局限包括：

- 场景规模较小；
- 物体类别有限；
- 任务结构较固定；
- 每类任务评估 episode 数量有限；
- 缺乏跨实验室复现验证。

### 8.2 对外部感知模型依赖较强

SOMA 的空间记忆质量依赖：

- YOLO 的检测准确率；
- DINOv3 的语义表征；
- VGGT 的位姿与几何估计。

如果这些模块在新环境中失效，例如：

- 透明物体；
- 反光物体；
- 纹理稀少物体；
- 高遮挡场景；
- 强动态场景；

则 SOMA 的记忆 token 可能噪声较大，进而误导动作生成。

论文的失败分析也指出，VGGT 几何噪声是仿真失败的重要原因之一。

### 8.3 动态环境处理能力有限

论文假设扫描阶段是短时、准静态环境。  
如果场景中人或其他机器人持续移动物体，当前方法可能面临：

- 记忆过期；
- 物体关联错误；
- 历史位置误导动作；
- 需要频繁重新扫描。

作者也承认未来需要：

- 实时 tracking；
- confidence-based re-scanning；
- memory expiration；
- 更强的动态场景建模。

### 8.4 当前记忆粒度主要是物体级

SOMA 的记忆主要表示物体的外观和 3D 位置，但对任务阶段和状态变化建模不足。

例如：

- 抽屉是开还是关；
- 容器是否已被放入物体；
- 某个阶段是否已经完成；
- 物体是否被另一只手持有。

论文失败分析中也提到，缺乏 task-phase awareness 是多阶段任务中的失败来源之一。

### 8.5 视角运动下的精确控制仍然困难

真实失败案例中，最大失败类型是动态视角下抓取姿态不准确。

这说明：

- SOMA 能提供较可靠的空间定位；
- 但从空间记忆到精确抓取控制之间仍有误差；
- 头部相机持续运动时，视觉定位和机械臂控制之间的校准更困难。

未来可能需要：

- 更强的头-臂协同控制；
- viewpoint-stabilized action decoding；
- 基于闭环视觉伺服的精细抓取。

### 8.6 推理延迟有所增加

SOMA 的在线推理延迟约为 1.58 秒 / action chunk，高于 GR00T-N1.5 和 StarVLA。

虽然仍可接受，但对于高速操作、动态避障或人机协作场景，延迟可能成为限制。

### 8.7 安全与隐私问题

持久空间记忆会保存环境中的物体和布局信息，应用于家庭或工业环境时可能涉及：

- 隐私泄露；
- 过期记忆导致错误操作；
- 人类共享空间中的安全风险；
- 被误用为环境监控系统。

论文也提到需要考虑：

- 置信度评分；
- 重新扫描触发；
- 记忆过期机制；
- 本地存储；
- 人类确认机制。

# 总体评价

SOMA 是一篇针对 VLA 机器人操作中“视野外目标”问题的系统性工作。它的核心贡献不是提出更大的 VLA 模型，而是引入一种持久、可更新、可检索的空间语义记忆机制，使机器人能够超越当前相机视野进行操作决策。

论文的实验覆盖真实机器人、仿真 benchmark、行为分析、消融实验和失败分析，整体较完整。其主要价值在于指出并验证了一个重要方向：

> 对于真实世界机器人操作，空间记忆可能是 VLA 从短视反应式策略走向长期、稳定、主动操作的重要组件。

不过，该方法仍依赖强感知模块和相对受控的桌面环境，对动态开放场景、状态级任务理解、精确闭环控制和安全机制仍需进一步研究。
