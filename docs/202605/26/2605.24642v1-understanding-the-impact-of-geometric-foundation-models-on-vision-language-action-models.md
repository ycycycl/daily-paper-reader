## 摘要
近期研究探索了视觉-语言-动作模型（VLA）与用于三维重建的几何基础模型（GFM）（如 VGGT）交叉领域的新机遇。尽管由此得到的几何 VLA 往往表现出更好的性能，但以下问题仍不明确：（i）现代 VLA 本身是否已经具备足够的几何理解能力；（ii）将几何理解注入 VLA 的最佳架构是什么；（iii）影响几何 VLA 的其他设计选择会产生什么影响。本文针对特定的 VLA（GR00T-N1.5）和 GFM（VGGT）选择，提供了严格的实验分析，以阐明这些问题。我们的第一项贡献是通过基于线性探测的严谨分析，将先前工作中关于当前 VLA 缺乏几何理解的直觉形式化。该分析首次量化了 VLA 与 GFM 之间的“几何差距”。我们的第二项贡献是识别并比较将 GFM 与 VLA 连接起来的不同策略。我们实现了三种不同架构，它们在向 VLA 注入几何信息的方式上有所不同，同时尽可能保持底层实现细节相似，以确保公平比较。最后，我们分析了非架构选择（例如训练数据、相机数量、重建质量）对几何 VLA 性能的影响。

## Abstract
Recent work explores new opportunities at the intersection of vision-language-action models (VLAs) and geometric foundation models (GFMs) for 3D reconstruction, such as VGGT. While the resulting geometric VLAs often show improved performance, it remains unclear (i) if modern VLAs already have sufficient geometric understanding to start with, (ii) what is the best architecture to inject geometric understanding into a VLA, and (iii) what is the effect of other design choices that affect geometric VLAs. In this paper we provide a rigorous experimental analysis to shed light on these questions, for a specific choice of VLA (GR00T-N1.5) and GFM (VGGT). Our first contribution is to formalize prior work's intuition that current VLAs lack geometric understanding, by providing a rigorous analysis based on linear probing. The analysis quantifies, for the first time, the "geometric gap" between VLAs and GFMs. Our second contribution is to identify and compare different strategies to bridge GFMs with VLAs. We implement three different architectures, which differ in the way they inject geometry in the VLA, while keeping low-level implementation details as similar as possible, to ensure a fair comparison. Finally, we analyze the impact of non-architectural choices (e.g., training data, number of cameras, reconstruction quality) on the performance of the geometric VLAs.

## 2. 方法论：核心思想、架构与技术细节

本文以 **GR00T-N1.5** 作为 VLA，以 **VGGT** 作为 GFM，系统比较几种将几何信息注入 VLA 的策略。

### 2.1 标准 VLA 架构

标准 VLA 输入包括：

- 图像集合 \(I\)；
- 语言指令 \(L\)；
- 机器人状态 \(R\)。

处理流程如下：

1. 图像经过视觉编码器，得到视觉 token \(V_e\)；
2. 文本经过 tokenizer / 文本编码器，得到语言 token \(L_e\)；
3. 机器人状态经过状态编码器，得到状态 token \(R\)；
4. 视觉和语言 token 输入 LLM / VLM backbone，输出：
   - 新的视觉 token \(V_l\)；
   - 新的语言 token \(L_l\)；
5. 动作专家根据 \(V_l, L_l, R\) 预测未来一段动作序列。

GR00T 的动作专家采用扩散策略 / flow matching 形式。训练时对真实动作序列加入噪声，并训练模型预测从噪声动作恢复真实动作的去噪场。

### 2.2 三种几何注入策略

论文总结并实现了三类几何注入方法。

#### 方法一：Early Fusion

**核心思想**：在 VLA 的视觉 token 进入 LLM 之前，就将 VGGT 几何 token 注入进去。

流程：

1. VLA 视觉编码器从图像得到视觉 token \(V_e\)；
2. VGGT 从同一批图像得到几何 token \(G\)；
3. 使用 cross-attention 融合 \(V_e\) 与 \(G\)；
4. 融合后的 token 替代原始视觉 token 输入 LLM。

直观上，Early Fusion 把 VGGT 当作一个额外几何编码器，让 LLM 有机会在较早阶段处理几何信息。

#### 方法二：Late Fusion

**核心思想**：先让 VLA 的 VLM 正常处理视觉和语言信息，在 VLM 输出后、进入动作专家前，再注入 VGGT 几何 token。

流程：

1. 图像和语言先进入 GR00T 的 VLM；
2. 得到 VLM 输出的视觉 token \(V_l\)；
3. VGGT 同时输出几何 token \(G\)；
4. 将 \(V_l\) 与 \(G\) 通过 cross-attention 融合；
5. 融合后的 token 输入动作专家。

直观上，Late Fusion 试图直接增强动作专家看到的表征，使其包含更多几何信息。

#### 方法三：Spatial Forcing

**核心思想**：不改变推理时的 VLA 架构，而是在训练时增加一个对齐损失，让 VLA 内部 token 向 VGGT 几何表征靠近。

具体做法：

- 在 GR00T 的 LLM 某一中间层加入 alignment loss；
- 用余弦相似度约束 VLA 表征与 VGGT token 对齐；
- 目的是让 VLA 自身内部表征保留更多几何信息。

论文中将 Spatial Forcing 的对齐损失施加在 GR00T LLM 的第 9 层。

### 2.3 Cross-Attention Fusion 机制

Early Fusion 和 Late Fusion 都使用类似的融合模块。

设：

- VLA token 为 \(X\)，其中 Early Fusion 中 \(X=V_e\)，Late Fusion 中 \(X=V_l\)；
- VGGT 几何 token 为 \(G\)。

融合过程为：

1. 将 VLA token 映射为 query：
   \[
   Q = X W_Q
   \]

2. 将 VGGT token 映射为 key 和 value：
   \[
   K = G W_K,\quad V = G W_V
   \]

3. 做 cross-attention：
   \[
   Y = \text{softmax}\left(\frac{QK^\top}{\sqrt{d}}\right)V
   \]

4. 将结果投影回 VLA token 空间：
   \[
   Z = Y W_O
   \]

5. 使用 attention gate 作为残差修正：
   \[
   \tilde{X} = X + A \odot Z
   \]

其中 \(A\) 是可学习门控，初始化接近 0。

### 2.4 Attention Gate 的作用

论文发现 attention gate 非常关键：

- 如果直接把 VGGT token 注入 VLA，可能让动作专家分布外，导致训练不稳定；
- 将 gate 初始化接近 0，可以让几何信息逐渐进入 VLA；
- 消融实验显示，不使用 attention gate 的 Early Fusion 表现明显变差。

### 2.5 线性探测：量化“几何差距”

为了回答 VLA 是否本身具备几何理解，论文使用 **linear probing**。

做法：

- 冻结 GR00T 或 VGGT；
- 在某一层 token 后接一个简单 MLP probe；
- 训练 probe 预测深度图；
- 如果 probe 可以较好预测深度，说明该层 token 含有几何信息；
- 如果不能，则说明几何信息缺失。

评估指标包括：

- RMSE depth error，越低越好；
- \(\delta_1\)，即预测深度在真实深度 25% 误差范围内的像素比例，越高越好。

## 6. 主要结论与发现

### 6.1 当前 VLA 确实存在明显“几何差距”

线性探测结果显示：

| 表征来源 | RMSE ↓ | δ1 ↑ |
|---|---:|---:|
| GR00T vision encoder probe | 0.92 | 0.51 |
| GR00T VLM probe | 0.73 | 0.63 |
| VGGT probe | 0.41 | 0.89 |
| Early Fusion probe | 0.44 | 0.88 |
| Late Fusion probe | 0.45 | 0.87 |

这说明：

- GR00T 内部表征中的深度信息明显弱于 VGGT；
- 几何信息在 VLA 中并未被充分保留；
- VGGT token 可以显著补足这种几何信息缺失。

### 6.2 Early Fusion 和 Late Fusion 都能注入几何信息

- Late Fusion 能保留几何信息并不意外，因为 VGGT token 被直接注入到接近 action expert 的位置；
- 更令人意外的是，Early Fusion 在 LLM 冻结的情况下仍能让模型输出接近 VGGT 的深度预测表现；
- 这说明 GR00T 的 LLM 对新几何 token 具有一定“可塑性”。

### 6.3 简单 task-level finetuning 下，几何 VLA 不一定显著优于 baseline

在 RoboCasa 标准多相机设置中：

- GR00T-N1.5 平均成功率约 71.7%；
- Early Fusion 约 69.7%；
- Late Fusion 约 71.0%；
- Spatial Forcing 约 68.3%。

这些差异大多不具统计显著性。

结论是：

- 仅在单任务小数据上 finetune，加入 VGGT 不一定能显著提升操作性能；
- 几何信息存在，但 action policy 未必能充分利用。

### 6.4 扩大训练数据后，Early Fusion 优势更明显

在 RoboCasa 上进行 mid-training 后：

- Early Fusion 平均成功率提升到 75.2%；
- 相比 GR00T-N1.5 baseline 有更明显优势；
- 与 mid-trained GR00T 相比，Early Fusion 在部分任务上仍有优势。

这说明：

- 几何 token 的使用需要足够训练数据让模型适应；
- scaling training data 可能是释放几何 VLA 潜力的关键。

### 6.5 单相机设置下，VGGT 更有价值

在只使用单相机时：

- GR00T-N1.5 平均成功率：17.2%；
- Early Fusion：21.5%；
- 差异具有统计显著性。

这说明：

- 多相机输入本身已经提供一定几何线索；
- 当传感器受限时，GFM 提供的几何先验更重要。

### 6.6 VGGT 重建质量与操作成功率存在相关性

论文发现：

- VGGT 深度误差越低，操作成功率整体倾向越高；
- Spearman 相关系数为 \(\rho = -0.202\)，表示轻微负相关。

这说明：

- GFM 重建质量会影响几何 VLA 表现；
- 在目标环境中进一步微调 VGGT 可能继续提升性能。

### 6.7 真实机器人上 Early Fusion 更有潜力

Unitree G1 实验中：

- GR00T-N1.5 overall success：22.22%；
- Early Fusion：27.78%；
- Late Fusion：25.56%。

总体提升未达到强显著性，但 Early Fusion 在 Approach 阶段显著更好：

- GR00T-N1.5：57.78%；
- Early Fusion：84.44%。

这暗示 Early Fusion 可能更擅长利用几何信息帮助机器人接近和定位目标物体，尤其是小物体。

## 7. 优点与亮点

### 7.1 问题定义清晰

论文没有简单提出一个新架构，而是围绕三个关键问题展开：

- VLA 是否缺几何？
- 如何注入几何？
- 哪些因素影响几何注入效果？

这种问题导向非常明确。

### 7.2 首次系统量化 VLA 与 GFM 的“几何差距”

通过 linear probing，论文将“VLA 缺乏几何理解”从直觉转化为可量化结果。

这比单纯报告机器人成功率更有解释力。

### 7.3 架构比较较公平

论文实现了 Early Fusion、Late Fusion、Spatial Forcing 三类方法，并尽量统一底层实现，例如：

- 都基于 GR00T-N1.5；
- 都使用 VGGT；
- Early / Late Fusion 使用同样的 cross-attention fusion module；
- 尽量冻结相同组件，避免参数量和训练策略差异造成混淆。

### 7.4 重视统计显著性

论文没有仅凭平均成功率下结论，而是使用 McNemar’s test 分析 paired binary success/failure。

这在机器人学习论文中是一个重要优点，因为 VLA 策略的成功率波动很大。

### 7.5 对随机性进行了专门分析

附录中展示了即使固定场景，diffusion action expert 的随机性也会导致成功率明显波动。

这使论文对实验结论更加谨慎，也提高了可信度。

### 7.6 覆盖仿真和真实机器人

除了 RoboCasa 和 LIBERO，论文还在 Unitree G1 上进行真实实验，使结论不完全局限于仿真。

### 7.7 分析了非架构因素

论文不仅比较模型结构，还分析了：

- 训练数据规模；
- 相机数量；
- VGGT 重建误差；
- 物体外观变化；
- 表面法向预测能力。

这让研究更接近系统性评估，而不是单点性能比较。

## 8. 不足与局限

### 8.1 结论依赖特定模型组合

论文主要基于：

- VLA：GR00T-N1.5；
- GFM：VGGT。

因此结论未必能直接推广到：

- OpenVLA；
- π0；
- RT-1 / RT-2 系列；
- 其他 GFM，如 DUSt3R、MASt3R、MapAnything 等。

### 8.2 仿真实验占多数

虽然有 Unitree G1 实验，但主要结果仍来自 RoboCasa 和 LIBERO。

仿真环境可能存在：

- 视觉渲染偏差；
- 物理交互简化；
- 任务分布较规则；
- 与真实世界接触动力学不完全一致。

### 8.3 真实机器人实验规模有限

Unitree G1 实验共有 90 次，覆盖 3 类物体。

虽然实验成本高可以理解，但从统计角度看：

- 任务类型较少；
- 场景变化有限；
- overall success 的提升未达到强显著性；
- 还不足以证明真实世界中几何 VLA 一定显著优于 baseline。

### 8.4 部分 benchmark 可能接近饱和

在 LIBERO 上，GR00T-N1.5 baseline 已经有较高成功率，例如部分子任务超过 95%。

这会导致：

- 提升空间有限；
- 几何增强的优势难以体现；
- benchmark 对几何能力的区分度不足。

### 8.5 没有穷尽所有架构选择

论文承认没有系统探索全部设计空间，例如：

- Spatial Forcing 应施加在哪一层；
- 是否使用其他 fusion 方式；
- 是否更深层融合；
- 是否让 VGGT 输出显式深度图或点云，而非 token；
- 是否联合训练 VGGT；
- 是否设计更强的 action expert 来利用几何 token。

### 8.6 Action expert 可能是瓶颈

论文观察到 Late Fusion 效果不如预期，推测 action expert 对新信息较“刚性”。

这提示当前 VLA 的动作模块可能未能充分利用几何 token。也就是说，几何信息虽然注入了，但未必能有效转化为动作决策。

### 8.7 GFM 错误会传播到策略

VGGT 虽然泛化强，但仍可能在分布外场景中出错。

如果 GFM 预测的深度或结构错误，几何 VLA 可能受到误导。这在安全关键机器人任务中尤其需要注意。

### 8.8 算力成本较高

普通 finetuning 需要 2–3 天，mid-training 约 1 周，并使用 320GB A100 GPU 集群显存。

这说明该类实验和训练对资源要求不低，限制了普通实验室复现和扩展。

---

## 总结

本文系统研究了几何基础模型对 VLA 的影响。核心发现是：**现有 VLA 确实缺乏稠密几何理解，而 VGGT 等 GFM 可以显著补足这种表征缺口；但几何信息能否转化为机器人操作性能提升，取决于训练数据规模、相机配置、融合架构和 GFM 重建质量等因素。**

在标准多相机、小规模 task-level finetuning 下，几何 VLA 不一定显著优于 GR00T-N1.5；但在 mid-training、单相机或真实机器人场景中，Early Fusion 展现出更明显潜力。论文的价值主要在于提供了一个严谨、统计化、系统性的评估框架，而不是单纯追求平均成功率提升。
