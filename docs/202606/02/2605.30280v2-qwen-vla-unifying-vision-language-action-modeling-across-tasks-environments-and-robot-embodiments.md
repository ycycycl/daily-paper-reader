## 摘要
具身智能通常通过针对操作或导航等单一任务的专门模型进行研究，这导致能力碎片化，并限制了其在任务、环境和机器人具身形态之间的泛化能力。在本文中，我们研究异构的具身决策问题是否能够在单一的视觉-语言-动作模型中得到统一。我们提出 Qwen-VLA，这是一种统一的具身基础模型，通过基于 DiT 的动作解码器，将 Qwen 的视觉-语言建模技术栈从感知、理解与推理扩展到连续动作和轨迹生成。Qwen-VLA 采用大规模联合预训练方案，在多样化数据源上进行训练，包括机器人操作轨迹、人类第一视角示范、合成仿真数据、视觉-语言导航数据、以轨迹为中心的监督数据以及辅助视觉-语言数据。为支持多种机器人平台，我们引入了具身感知的提示条件机制，其中机器人特定的文本描述用于指定当前具身形态和控制约定。我们进一步将操作、导航和轨迹预测表述为统一的动作与轨迹预测框架，从而在不同机器人形态、任务族和环境之间实现可迁移的视觉定位、空间推理和连续动作生成。在操作、导航和以轨迹为中心的基准上的实验表明，在场景布局、背景、光照、物体配置和机器人具身形态发生变化时，该模型具有一致的多任务性能和分布外泛化能力。Qwen-VLA-Instruct 在 LIBERO 上达到 97.9%，在 Simpler-WidowX 上达到 73.7%，在 RoboTwin-Easy/Hard 上达到 86.1%/87.2%，在 R2R 上达到 69.0% OSR，在 RxR 上达到 59.6% SR，在真实世界 ALOHA 实验中达到 76.9% 的平均 OOD 成功率，并在 DOMINO 动态操作中实现 26.6% 的零样本成功率。

## Abstract
Embodied intelligence is often studied through specialized models for individual tasks such as manipulation or navigation, resulting in fragmented capabilities and limited generalization across tasks, environments, and robot embodiments. In this work, we study whether heterogeneous embodied decision-making problems can be unified within a single vision-language-action model. We present Qwen-VLA, a unified embodied foundation model that extends Qwen's vision-language modeling stack from perception, understanding, and reasoning to continuous action and trajectory generation through a DiT-based action decoder. Qwen-VLA is trained with a large-scale joint pretraining recipe over diverse data sources, including robotics manipulation trajectories, human egocentric demonstrations, synthetic simulation data, vision-and-language navigation data, trajectory-centric supervision, and auxiliary vision-language data. To support multiple robot platforms, we introduce embodiment-aware prompt conditioning, where robot-specific textual descriptions specify the current embodiment and control convention. We further cast manipulation, navigation, and trajectory prediction into a unified action-and-trajectory prediction framework, enabling transferable visual grounding, spatial reasoning, and continuous action generation across robot morphologies, task families, and environments. Experiments on manipulation, navigation, and trajectory-centric benchmarks show consistent multi-task performance and out-of-distribution generalization under variations in scene layout, background, lighting, object configuration, and robot embodiment. Qwen-VLA-Instruct achieves 97.9% on LIBERO, 73.7% on Simpler-WidowX, 86.1%/87.2% on RoboTwin-Easy/Hard, 69.0% OSR on R2R, 59.6% SR on RxR, 76.9% average OOD success in real-world ALOHA experiments, and 26.6% zero-shot success on DOMINO dynamic manipulation.

## 2. 方法论：核心思想、关键技术细节与训练流程

## 2.1 统一问题表述

论文将多种具身任务统一为条件动作 / 轨迹预测问题：

\[
p_\theta(y_{t:t+H-1} \mid o_t, x, e, z)
\]

其中：

- \(o_t\)：视觉上下文，可以是单帧图像、多视角图像、视频或历史窗口；
- \(x\)：语言指令；
- \(e\)：具身形态描述，例如机器人平台、控制频率、动作空间、预测 horizon；
- \(z\)：可选任务类型标识；
- \(y_{t:t+H-1}\)：未来动作或轨迹序列；
- \(H\)：预测时间跨度。

这一公式将不同任务统一为“给定视觉、语言和具身约束，预测未来动作或轨迹”。

## 2.2 模型架构

Qwen-VLA 由两部分组成：

### 1. 视觉语言骨干网络

- 使用 **Qwen3.5-4B** 作为多模态 backbone；
- 具备图像、视频、文本联合建模能力；
- 支持早期视觉语言融合；
- 负责：
  - 视觉感知；
  - 目标识别；
  - 指令理解；
  - 空间关系推理；
  - 多语言和结构化推理。

### 2. DiT-based Flow-Matching 动作专家

- 在 VLM 后接入一个基于 **Diffusion Transformer / DiT** 的动作解码器；
- 参数规模约 **1.15B**；
- 使用 flow matching 目标生成连续动作；
- 输入包括：
  - VLM hidden states；
  - noisy action chunk；
  - timestep embedding；
- 输出为 clean action chunk；
- 推理时通过少量 Euler integration steps 从噪声生成动作序列，支持低延迟控制。

## 2.3 具身形态感知提示 Embodiment-aware Prompt

论文不为不同机器人设计不同 head，而是用文本提示描述机器人控制约定。

模板大致为：

> The robot is {robot_tag} with {single arm / dual arms}[, waist][, and mobile base].  
> The control frequency is {FPS} Hz.  
> Please predict the next {chunk_size} control actions to execute the following task: {instruction}.

提示中包含：

- 机器人名称；
- 单臂 / 双臂 / 腰部 / 移动底盘；
- 控制频率；
- 动作 horizon；
- 原始语言任务。

这种设计让模型通过文本接口理解不同机器人平台和动作语义。

## 2.4 统一动作与轨迹表示

论文没有强行把所有动作转成同一种物理语义，而是统一张量接口和 mask 机制。

- 每个样本动作表示为：

\[
Y \in \mathbb{R}^{H \times K}
\]

其中：

- \(H\)：统一预测 horizon；
- \(K\)：统一最大动作通道数；
- 不同任务使用前 \(c \leq K\) 个通道；
- 未使用通道补零；
- 通过 mask \(M\) 屏蔽 padding 部分。

支持的动作类型包括：

- 操作任务：
  - 末端执行器增量位姿；
  - 末端旋转；
  - 绝对关节角；
  - 夹爪状态；
  - 灵巧手关节；
- 导航任务：
  - \((\Delta x, \Delta y, \Delta \theta)\) waypoint；
- 人类第一视角动作：
  - 双手 wrist motion；
  - MANO / hand pose；
  - eigengrasp 低维手部表示。

## 2.5 损失函数

训练目标包括两类：

### 1. Flow-matching action loss

- 对连续动作目标进行监督；
- 从 clean action \(Y_0\) 和噪声 \(Y_1\) 构造插值：

\[
Y_\tau = (1-\tau)Y_0 + \tau Y_1
\]

- 模型预测速度场：

\[
v_\theta(Y_\tau, \tau | o, x, e, z)
\]

- 使用 mask 避免 padding 通道影响梯度；
- 对每个有效动作通道做均衡平均，防止高维动作平台主导训练。

### 2. Vision-language loss

- 保留标准 next-token prediction；
- 用于视觉问答、caption、grounding、自动驾驶 VQA、通用 VL 数据；
- 目的：
  - 防止 VLM backbone 在动作训练中遗忘视觉语言能力；
  - 保持语言 grounding 和视觉推理能力。

### 3. 总损失

\[
L = \lambda_{act} L_{act} + \lambda_{vl} L_{vl}
\]

其中 \(\lambda_{act}\) 和 \(\lambda_{vl}\) 用于平衡动作学习和视觉语言学习。

## 2.6 四阶段训练流程

论文提出渐进式训练策略：

### Stage I：Text-to-Action DiT Pretraining, T2A

- 冻结 VLM；
- 只训练 DiT 动作解码器；
- 不输入图像，只输入语言指令和具身提示；
- 目标是让动作解码器先学习“语言到动作轨迹”的结构化先验；
- 作者将其解释为“从压缩语言描述到高维动作轨迹的解压缩问题”。

### Stage II：Continued Pretraining, CPT

- 解冻 VLM 和 DiT；
- 使用大规模异构数据进行多模态联合预训练；
- 让 T2A 学到的语言动作先验进一步视觉 grounding。

### Stage III：Supervised Fine-Tuning, SFT

分为两个方向：

- 多任务 SFT：
  - VQA；
  - spatial grounding；
  - 操作；
  - 导航；
- 真实机器人 SFT：
  - 基于 ALOHA 等真实机器人遥操作数据微调。

### Stage IV：Reinforcement Learning, RL

- 从 SFT checkpoint 开始；
- 使用 PPO + GAE；
- 奖励为模拟器中的稀疏二值任务成功奖励；
- 只在 SimplerEnv 中进行 RL；
- 最终得到 **Qwen-VLA-Instruct**。

## 4. 资源与算力

  - EgoDex 829 小时；
  - EgoVerse 1,300 小时。

- **但论文没有明确报告完整训练算力配置**，例如：
  - 使用多少 GPU；
  - GPU 型号；
  - 训练总时长；
  - 总 FLOPs；
  - batch size；
  - 全阶段训练成本。

- 论文只在合成数据生成和 RL 基础设施中提到：

## 6. 主要结论与发现

- **统一建模可行**：  
  操作、导航、轨迹预测和人类动作建模可以统一为动作与轨迹预测问题。

- **单一 generalist 模型可以接近甚至超过 specialist 模型**：  
  Qwen-VLA-Instruct 在多个操作 benchmark 上超过或接近专用模型。

- **大规模预训练显著提升真实机器人性能**：  
  在 ALOHA 上，Qwen-VLA-aloha w/ pretrain 平均 in-domain 成功率达到 83.6%，而从头训练只有 48.5%。

- **预训练提升 OOD 泛化能力**：  
  ALOHA OOD 平均成功率从 36.2% 提升到 76.9%。

- **T2A 阶段有效**：  
  语言到动作预训练能为 DiT 建立结构化动作先验，在 Simpler-WidowX 上相比无 T2A 提升明显。

- **VL 数据对动作学习有帮助而非干扰**：  
  在复杂 benchmark 上，加入视觉语言数据提高了物体识别和组合指令理解能力。

- **RL 后训练有增益但幅度相对有限**：  
  RL 主要提升 SimplerEnv，并在其他 benchmark 上保持或略微提升性能，没有明显灾难性遗忘。

- **显式 proprioceptive state 不是必要条件**：  
  状态输入只带来很小提升，因此论文选择默认不使用 state conditioning，以保持跨具身形态接口简洁。

## 7. 优点与亮点

- **统一性强**：
  - 将操作、导航、人类第一视角动作、轨迹预测统一到一个条件动作预测框架；
  - 避免为每个任务或机器人单独设计模型。

- **具身形态提示设计简洁有效**：
  - 用自然语言描述机器人平台和控制约定；
  - 不需要每个机器人单独 head；
  - 有利于跨平台扩展。

- **DiT + flow matching 适合连续动作生成**：
  - 能生成高维连续动作 chunk；
  - 推理时使用少量 Euler steps，兼顾精度和速度。

- **训练流程设计合理**：
  - T2A 学动作先验；
  - CPT 做视觉 grounding；
  - SFT 做任务对齐；
  - RL 优化闭环成功率；
  - 每个阶段目标清晰。

- **数据规模和多样性大**：
  - 真实机器人；
  - 仿真；
  - 人类第一视角；
  - 导航；
  - 自动驾驶 VQA；
  - 空间 grounding；
  - 通用视觉语言数据。

- **实验覆盖面广**：
  - 操作、导航、真实机器人、OOD、动态 manipulation 都有评估；
  - 消融实验较系统。

- **真实机器人 OOD 结果突出**：
  - 对颜色、物体实例、背景、位置、指令变化均有测试；
  - 能较好说明预训练带来的泛化能力。

- **对自动驾驶 / 轨迹预测方向有启发**：
  - 论文将导航 waypoint、操作 action、人类 motion 都视为连续轨迹预测；
  - 对上游规划、trajectory generation、multi-agent motion prediction 等方向也有借鉴价值。

## 8. 不足与局限

- **训练成本不透明**：
  - 未报告 GPU 型号、数量、总训练时长、总 FLOPs；
  - 对复现和成本评估不利。

- **内部数据依赖较强**：
  - 包含大量内部真实机器人数据和合成数据；
  - 外部研究者难以完整复现训练集和结果。

- **数据规模与 baseline 不完全对等**：
  - Qwen-VLA 使用非常大规模和多样化数据；
  - 与部分专用模型比较时，胜出可能部分来自数据规模优势。

- **长时程任务能力仍有限**：
  - 论文承认当前评估仍以短时程和 benchmark-driven 任务为主；
  - 对真实长期部署、失败恢复、任务重规划的验证不足。

- **接触丰富任务仍是挑战**：
  - 对力控、触觉、复杂接触、柔性物体长期操作的覆盖有限；
  - Towel folding 虽有评估，但整体接触物理能力仍未充分展开。

- **世界模型和记忆机制不足**：
  - 当前主要是动作 / 轨迹生成；
  - 没有显式长期记忆、环境状态跟踪或未来视觉预测；
  - 对长程规划和失败恢复可能不足。

- **RL 范围较窄**：
  - RL 只在 SimplerEnv 中进行；
  - 虽然结果显示有一定跨域迁移，但仍不足以证明大规模 RL 对所有环境都有效。

- **导航能力虽强但可能受操作训练牵制**：
  - 作者在局限中提到，多任务联合训练可能导致纯 VL 或导航指标有一定回退；
  - 如何平衡 VL、VLN、VLA 目标仍是开放问题。

- **真实世界评估规模仍有限**：
  - ALOHA 实验有价值，但平台单一；
  - 尚未充分证明在更多真实机器人、更多家庭 / 工业环境中的稳定部署能力。

# 总体评价

Qwen-VLA 是一篇面向通用具身智能基础模型的重要工作。它的核心价值在于提出并实证验证了一个统一的视觉-语言-动作建模框架：通过 Qwen3.5 VLM backbone、DiT flow-matching 动作解码器、具身形态感知提示、多源联合预训练和 SFT/RL 后训练，将操作、导航和轨迹预测纳入同一个模型接口。实验结果显示，该模型在多个仿真和真实机器人 benchmark 上具有较强性能和泛化能力。

不过，论文也存在典型大模型系统论文的问题：训练资源披露不足、内部数据依赖强、复现难度较高、baseline 数据规模公平性不完全明确。总体而言，Qwen-VLA 在方法统一性、实验广度和跨具身泛化方面贡献突出，但仍需要更多开放数据、开放训练细节和长期真实部署验证来进一步巩固其结论。
