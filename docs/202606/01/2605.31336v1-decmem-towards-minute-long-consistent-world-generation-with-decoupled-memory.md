## 摘要
视频生成模型的最新进展推动了可控世界模型的快速发展。然而，在长时域推理下保持细粒度的时空一致性仍然是一项关键挑战。在本文中，我们超越显式三维记忆和粗粒度帧级隐式建模，提出了一种用于一致世界生成的细粒度、可学习且可扩展的记忆机制。我们首先指出朴素可学习记忆架构在长时域外推中的两个根本局限，即计算效率低下和注意力分散。通过对注意力分散的系统分析，我们提出了 DecMem，一种解耦记忆架构，它采用稀疏全局记忆以高效地细粒度访问全局历史，并采用锚定局部记忆以实现稳定且高质量的外推。大量实验表明，DecMem 显著优于当前最先进的方法。通过确保精确且高效的长期记忆，并实现卓越的外推能力，DecMem 能够生成具有高保真度和一致性的分钟级可控长视频。

## Abstract
Recent advances in video generative models have promoted rapid progress in controllable world models. However, maintaining fine-grained spatio-temporal consistency under long-horizon reasoning remains a key challenge. In this work, we move beyond explicit 3D memory and coarse frame-level implicit modeling, and propose a fine-grained, learnable, and scalable memory for consistent world generation. We first identify two fundamental limitations of naïve learnable memory architectures in long-horizon extrapolation, namely computational inefficiency and attention dispersion. Through a systematic analysis of attention dispersion, we propose DecMem, a decoupled memory architecture that employs Sparse Global Memory for efficient fine-grained access to global history and Anchored Local Memory for stable and high-quality extrapolation. Extensive experiments demonstrate that DecMem significantly outperforms current state-of-the-art methods. By ensuring precise and efficient long-term memory and achieving superior extrapolation capabilities, DecMem enables minute-level controllable long video generation with high fidelity and consistency.

## 2. 方法论：DecMem 解耦记忆架构

论文提出 **DecMem（Decoupled Memory）**，一种面向长视频世界生成的解耦记忆机制。其核心思想是将记忆功能拆分为两个互补分支：

1. **Sparse Global Memory, SGM：稀疏全局记忆**
2. **Anchored Local Memory, ALM：锚定局部记忆**

二者分别解决长期记忆访问和短期稳定外推问题。

---

### 2.1 基础任务形式：自回归视频生成与动作控制

- 模型工作在 VAE 压缩后的 latent 空间中。
- 给定历史 latent 序列：

  \[
  z^0_{1:T}
  \]

  模型预测下一帧或下一 chunk 的 latent。

- 训练时采用 Rectified Flow 框架：
  - 对目标帧加入噪声；
  - 让模型预测从噪声到干净 latent 的 flow velocity。

- 损失形式为：

  \[
  L = \|v_\theta(z^0_{1:T}, z^t_{T+1}, t) - v_{T+1}\|^2_2
  \]

- 动作控制通过轻量映射模块注入 latent：

  \[
  x = Patchify(z) \oplus \psi(a)
  \]

  其中：
  - \(z\) 是视频 latent；
  - \(a\) 是动作控制信号；
  - \(\psi(a)\) 是动作嵌入；
  - \(\oplus\) 表示逐元素相加。

---

### 2.2 注意力分散问题分析

论文系统分析了长时推理中的 **attention dispersion**：

- 当历史帧越来越多时，当前 query 会对大量历史 token 分配小但非零的注意力权重。
- 这些弱相关 token 构成长尾注意力分布。
- 结果是：
  - 真正关键的历史区域获得的注意力被稀释；
  - 模型难以稳定回忆之前出现过的场景；
  - 长时外推时视频质量下降、结构崩溃。

论文还指出，训练无关的 temporal decay 方法虽然可以压低远距离 token 权重、提升短期质量，但会同时抑制真正有用的长期记忆，因此无法兼顾短期保真和长期一致性。

---

### 2.3 Sparse Global Memory：稀疏全局记忆

SGM 用于从完整历史中进行**可学习、细粒度、低成本的长期记忆检索**。

核心流程如下：

- **步骤 1：将每帧 latent 划分为多个非重叠 block**
  - 论文实现中每帧划分为 6 个 block。
  - 每个 block 内包含若干 token。

- **步骤 2：对每个 block 做 pooling，得到 block-level 表征**
  - 使用 pooled block feature 代表该局部区域。
  - 相比整帧检索，block 级别更细；
  - 相比 token 级全局 attention，计算更省。

- **步骤 3：当前帧每个 query block 与历史 block 计算相关性**
  - 对当前 block 的 pooled query 与历史 block 的 pooled key 计算注意力分数。
  - 选取得分最高的 top-k 个历史 block。
  - 默认设置中，top-k = 80。

- **步骤 4：在检索到的 block 内做 token 级 dense attention**
  - 对当前 query block 中每个 token，仅与检索到的历史 block 中 token 做注意力。
  - 这样兼顾：
    - 全局历史访问；
    - 局部细粒度匹配；
    - 计算效率。

SGM 的作用是：  
**从长历史中找到真正相关的局部记忆区域，而不是让所有历史 token 都参与 dense attention。**

---

### 2.4 Anchored Local Memory：锚定局部记忆

ALM 用于缓解长历史注意力导致的分散问题，并保证短期生成稳定。

- ALM 仅关注最近若干帧。
- 实现上采用 sliding-window attention。
- 论文默认使用最近 8 个 frame latents 作为局部窗口。

其设计动机是：

- 相邻帧通常与当前帧有最强视觉和语义相关性；
- 局部窗口注意力可以提供稳定、高置信度的注意力锚点；
- 防止模型在长历史检索中被远距离弱相关 token 干扰。

---

### 2.5 SGM 与 ALM 的门控融合

DecMem 将两个分支输出通过可学习 gate 融合：

\[
o_t = o^{alm}_t + G_t \odot o^{sgm}_t
\]

其中：

- \(o^{alm}_t\)：局部记忆分支输出；
- \(o^{sgm}_t\)：稀疏全局记忆分支输出；
- \(G_t\)：由当前帧特征预测的可学习门控；
- \(\odot\)：逐元素乘法。

这个设计使模型能够自适应平衡：

- 短期稳定性；
- 长期一致性；
- 计算效率；
- 细粒度历史回忆。

论文还提出扩展版视频 RoPE，将以下三类信息分别编码到不同 channel group：

### 5.3 消融实验充分性

论文做了多个关键消融：

- **去掉 SGM**
  - 模型退化为局部上下文生成器；
  - 长期一致性明显下降。

- **去掉 ALM**
  - 早期质量尚可；
  - 但超过约 600 帧后质量严重退化；
  - 说明局部锚定对缓解注意力分散非常关键。

- **Dense Attention**
  - 计算成本随历史长度增长；
  - 长时生成质量崩溃。

- **Dense Attention + Decay**
  - 可改善后期部分质量；
  - 但牺牲中期和长期记忆能力。

- **top-k 检索数量**
  - 比较 k=20、50、80、100；
  - 发现 k=80 综合最优；
  - k 过大反而可能引入冗余上下文，削弱全局和局部记忆互补性。

- **Action CFG**
  - 无 CFG 在短期 PSNR 更高；
  - 有 CFG 在长时外推时 FID 和 LPIPS 更稳定；
  - 说明 CFG 有助于长期生成稳定性。

### 5.4 公平性与客观性评价

整体而言，实验较充分，但仍存在一些需要注意的地方：

- **公平性较强之处**：
  - 主实验使用相同 WorldMem 数据集；
  - 扩散方法统一使用 20 步 denoising；
  - 对不同模型的初始化设置尽量遵循其原始机制；
  - 同时报告客观指标、用户研究和速度指标。

- **潜在不完全公平之处**：
  - DecMem 基于 1B 预训练视频模型，并使用 64 张 H200 训练，资源规模较大；
  - 部分 baseline 可能模型规模、训练数据、工程优化程度不同；
  - 工业模型对比中，由于输入协议不同，只能用单图初始化和用户研究，无法直接做严格数值比较；
  - 主定量实验主要集中在 Minecraft / WorldMem 风格场景，对真实世界复杂视频的泛化仍未充分量化。

## 6. 主要结论与发现

- **DecMem 能显著提升长视频世界生成的一致性和画质**。  
  在训练窗口内和长时外推阶段均优于现有方法。

- **注意力分散是长时世界模型失败的重要根源**。  
  长历史 dense attention 会让大量弱相关 token 稀释关键记忆。

- **单纯 temporal decay 不是理想方案**。  
  它虽然能缓解短期注意力扩散，但会无差别压低远距离信息，破坏真正有用的长期记忆。

- **SGM 对长期记忆至关重要**。  
  稀疏全局 block 检索能在降低计算成本的同时保留细粒度历史信息。

- **ALM 对外推稳定性至关重要**。  
  局部锚定能稳定注意力分布，防止长时推理中画面和结构崩溃。

- **SGM + ALM 的解耦组合是关键**。  
  仅有全局记忆或仅有局部记忆都不够，二者配合才能同时实现长期一致性和短期保真。

- **方法可支持分钟级视频生成**。  
  论文展示了 60 秒级别的可控视频生成，并在 revisit 场景中保持较好一致性。

---

## 7. 优点与亮点

- **问题分析深入**  
  论文不只是提出新模块，还专门分析了 dense attention 长时失败的根因，即 attention dispersion。

- **方法设计清晰且有针对性**  
  SGM 解决全局长期记忆访问问题，ALM 解决局部稳定性和注意力锚定问题，分工明确。

- **细粒度记忆机制优于帧级检索**  
  block-level retrieval 能比 FOV-based frame retrieval 更准确地匹配局部区域和历史细节。

- **计算效率较好**  
  相比 dense attention，SGM 的 top-k block 检索避免对全部历史 token 做 attention，使长时推理成本更可控。

- **实验覆盖较全面**  
  包括定量、定性、用户研究、消融、效率、工业模型比较和跨数据集可视化。

- **指标结果提升明显**  
  在 PSNR、LPIPS、FID、FPS 和用户偏好上均表现突出。

- **具有实际应用潜力**  
  对游戏生成、虚拟仿真、具身智能训练环境和交互式视频生成都有潜在价值。

---

## 8. 不足与局限

- **训练成本高**  
  使用 64 张 NVIDIA H200 训练约 7 天，且基于 1B 级预训练模型，复现门槛较高。

- **尚未达到实时生成**  
  论文明确指出其重点是记忆和外推一致性，而非通过蒸馏或系统优化实现实时推理。

- **主要实验集中在 Minecraft 风格环境**  
  虽然有 Context-as-Memory 额外验证，但主定量 benchmark 仍偏游戏合成环境，对真实世界复杂场景的量化泛化不足。

- **与工业模型对比不完全严格**  
  Matrix-Game 2.0 和 WorldPlay 的输入协议不同，不能直接使用视频 memory bank 初始化，因此只能通过单图初始化用户研究比较。

- **用户研究可能存在主观偏差**  
  尽管方法匿名随机展示，但用户偏好仍可能受视频内容、视觉风格和任务理解影响。

- **长期一致性仍依赖训练数据和动作轨迹分布**  
  如果遇到训练集中少见的复杂交互、动态对象变化或开放世界事件，效果尚不明确。

- **记忆机制仍是 block 级别而非对象级别**  
  论文未来工作也提到可进一步结合压缩全局记忆和细粒度对象级记忆，以提升长期一致性。

- **潜在滥用风险**  
  高一致性的长视频生成可能被用于伪造、欺诈或大规模虚假内容生成，论文也在 broader impacts 中提醒需要加强检测和溯源技术。
