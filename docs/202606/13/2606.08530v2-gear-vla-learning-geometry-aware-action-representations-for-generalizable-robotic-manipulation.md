## 摘要
视觉-语言-动作（Vision-Language-Action, VLA）模型在基准测试中取得了很强的性能，但在面对未见物体、背景变化以及不同机器人本体的真实世界部署中仍然表现不佳。我们认为，这源于缺乏统一的几何感知操作表示，使现有 VLA 容易受到低层轨迹监督、未对齐的 3D 特征以及本体差异的影响。为解决这一问题，我们提出 GEAR-VLA，一个用于学习统一几何感知动作表示以实现可泛化机器人操作的 VLA 框架。GEAR-VLA 采用由粗到细的动作学习，其中多源具身预训练先赋予 VLM 具身推理能力和离散动作理解能力，随后潜在动作 token 将动作语义连接到一个梯度解耦的 DiT 连续动作专家。该框架进一步执行语义对齐的 3D 集成：在冻结原有与 VLM 对齐的视觉通路的同时，将一个可训练的 3D 空间骨干网络与 VLA 表示对齐。为在不同机器人之间共享该表示，GEAR-VLA 采用本体规范化，其中本体感知状态和本体不变动作将机器人差异限制在低层接口。大量仿真和真实世界实验表明其具有很强的泛化能力：GEAR-VLA 在 LIBERO、零样本 LIBERO-Plus 和 RoboTwin 2.0 上达到最先进性能，在 AgileX 上取得 85.9% 的成功率，在预训练未见过的 LDT-01 本体上取得 81.0% 的成功率，并在包含 212 个未见物体、共 6,360 次试验的通用抓取基准上取得 90.1% 的成功率。代码和模型将在 https://github.com/babynabeauty/GEAR-VLA 发布。

## Abstract
Vision-Language-Action (VLA) models achieve strong benchmark performance but still struggle in real-world deployment with unseen objects, background shifts, and different robot embodiments. We argue that this stems from the lack of a unified geometry-aware manipulation representation, leaving existing VLAs vulnerable to low-level trajectory supervision, misaligned 3D features, and embodiment differences. To address this, we propose GEAR-VLA, a VLA framework for learning unified geometry-aware action representations for generalizable robotic manipulation. GEAR-VLA adopts coarse-to-fine action learning, where multi-source embodied pretraining equips the VLM with embodied reasoning and discrete action understanding before latent action tokens connect action semantics to a gradient-decoupled DiT continuous action expert. It further performs semantic-aligned 3D integration by aligning a trainable 3D spatial backbone with the VLA representation while freezing the original VLM-aligned visual pathway. To share this representation across robots, GEAR-VLA uses embodiment canonicalization, where embodiment-aware states and embodiment-invariant actions confine robot differences to the low-level interface. Extensive simulation and real-world experiments demonstrate strong generalization: GEAR-VLA achieves state-of-the-art performance on LIBERO, zero-shot LIBERO-Plus, and RoboTwin 2.0, reaches 85.9% success on AgileX and 81.0% on the pretraining-unseen LDT-01 embodiment, and obtains 90.1% success on a 6,360-trial universal grasping benchmark with 212 unseen objects. Code and models will be released at https://github.com/babynabeauty/GEAR-VLA.

### 1.1 背景：VLA 模型的潜力与现实部署瓶颈

视觉-语言-动作模型（Vision-Language-Action, VLA）试图将视觉感知、语言理解和机器人控制统一到一个端到端模型中，使机器人能够根据自然语言指令和视觉观测直接输出动作。近年来，OpenVLA、π0、π0.5、X-VLA、ACoT-VLA 等模型在仿真和部分真实机器人任务中表现较强，说明大规模视觉语言模型与机器人数据结合具有构建通用操作策略的潜力。

但论文指出，现有 VLA 模型虽然在标准 benchmark 上表现优秀，在真实世界部署时仍面临明显泛化不足，尤其体现在：

- **未见物体泛化不足**：面对训练中没有出现过的物体形状、材质、尺寸时，成功率下降。
- **背景与光照变化敏感**：场景背景、光照、相机视角变化会影响模型判断。
- **跨机器人本体迁移困难**：不同机器人具有不同自由度、关节结构、末端执行器、工作空间和状态表示，导致同一策略难以直接迁移。
- **缺乏稳定的 3D 几何理解**：2D VLM 虽有语义能力，但对抓取、折叠、翻转等操作所需的空间结构、深度关系和物体几何理解不足。
- **低层轨迹监督可能损害高层动作语义**：直接用连续轨迹训练 VLM，容易让模型学习“轨迹拟合”而非真正的具身推理和动作语义。

### 1.2 论文的核心问题

论文认为，当前 VLA 模型缺少一种统一的、可迁移的、几何感知的动作表示。这种缺失导致三类关键问题：

1. **动作表示问题**  
   现有动作 token 通常由低层轨迹量化得到，容易使 VLM 过度模仿轨迹，而不是学习高层动作意图和操作语义。

2. **空间表示问题**  
   深度图、3D 位置编码或空间 foundation model 的特征通常与 VLM 的语义 token 空间不对齐，直接注入可能破坏原本的视觉语言语义表示。

3. **本体表示问题**  
   现有跨机器人方法常使用机器人专用 action head、adapter 或 soft prompt，这可能把机器人身份信息混入共享策略表示中，降低跨本体迁移效率。

### 1.3 整体含义

GEAR-VLA 的核心目标是：  
**构建一种统一的几何感知动作表示，使 VLA 模型能够同时具备动作语义理解、3D 空间感知和跨机器人本体泛化能力。**

它的总体思想可以概括为：

> 先让 VLM 学会“理解动作和空间”，再用一个与 VLM 梯度解耦的连续动作专家执行控制；同时引入语义对齐的 3D 几何特征，并通过本体规范化把机器人差异限制在低层状态-动作接口中。

## 2. 方法论：核心思想、关键技术与流程

GEAR-VLA 由三项关键设计组成：

1. **粗到细动作学习（Coarse-to-Fine Action Learning）**
2. **语义对齐的 3D 几何集成（Semantic-Aligned 3D Integration）**
3. **本体规范化（Embodiment Canonicalization）**

整体输入包括：

- 多视角 RGB 图像；
- 自然语言指令；
- 机器人状态；
- 可选的目标掩码或空间指示信息。

输出为未来一段时间的动作 chunk，即连续控制轨迹。

---

### 2.1 粗到细动作学习

#### 核心思想

GEAR-VLA 不直接让 VLM 从一开始就回归连续动作，而是先进行大规模具身预训练，使 VLM 学会：

- 视觉语言理解；
- 空间 grounding；
- 任务规划；
- 轨迹推理；
- 指向与 affordance 理解；
- mask tracking；
- 离散动作语义；
- 视频中的动作动态。

然后再接入一个基于 DiT 的连续动作专家，将高层动作语义转化为可执行的连续机器人动作。

---

#### 阶段一：具身 VLM 预训练

论文将多种任务统一为自回归 token 预测任务，包括：

- 文本 QA；
- 空间 grounding；
- 任务规划；
- 轨迹推理；
- FAST action token；
- latent action ID；
- mask tracking 等。

训练目标为：

\[
L_{VLM} = - \sum_i \log p_\theta(y_i | y_{<i}, O, l, s)
\]

其中：

- \(O\)：多视角观测；
- \(l\)：语言指令；
- \(s\)：机器人状态；
- \(y_i\)：目标 token，可以是文本、grounding token、规划 token、FAST action token 或 latent action token。

---

#### FAST action token

对于有机器人动作标注的数据，论文使用 FAST tokenizer 将连续机器人轨迹离散化为 action token。  
这些 token 用于让 VLM 学习机器人动作模式，但仍保持自回归 token 预测形式。

---

#### latent action ID

对于没有机器人动作标签的人类视频或网页视频，论文训练了一个 **因果 VQ-VAE latent action tokenizer**。

该 tokenizer 的作用是：

- 从连续视频片段中提取动作诱导的视觉变化；
- 生成离散 latent action IDs；
- 让模型从无动作标注的视频中学习操作动态。

与只使用初始帧和最终帧的 latent action 方法不同，GEAR-VLA 的 tokenizer 建模一段连续视频中的短时动作动态，因此能保留更连续的动作语义。

实现细节包括：

- 视频帧以 5 Hz 采样；
- 动作流为 30 Hz；
- 每个连续帧变化提取 4 个 latent action codes；
- codebook 大小为 16，分为两组，每组 8 个 entry；
- 使用因果 masked cross-attention，防止未来信息泄漏；
- 训练完成后丢弃 decoder，仅保留 encoder、attention module 和 quantizer 用于 latent action ID 提取。

---

### 2.2 连续动作生成：梯度解耦的 DiT 动作专家

在 VLM 获得具身语义和离散动作理解后，GEAR-VLA 接入一个基于 DiT 的连续动作专家。

关键点是：

- DiT 不访问完整 VLM 表示；
- 只使用 VLM 产生的 latent action token 的 K/V cache；
- 连续动作损失不反向传播到 VLM backbone；
- 通过 stop-gradient 保护 VLM 已学到的语义和空间表示。

动作专家使用 flow matching 训练：

\[
L_{FM} = \mathbb{E}_{A,\epsilon,\tau}
\left[
\| v_\phi(A_\tau, \tau, sg(h_{la})) - (A - \epsilon) \|_2^2
\right]
\]

其中：

\[
A_\tau = \tau A + (1-\tau)\epsilon
\]

含义如下：

- \(A\)：目标连续动作 chunk；
- \(\epsilon\)：高斯噪声；
- \(\tau\)：flow time；
- \(h_{la}\)：latent action token cache；
- \(sg(\cdot)\)：stop-gradient；
- \(v_\phi\)：DiT 动作专家预测的 velocity field。

这种设计使连续控制不再是简单的 VLM 回归头，而是通过 latent action token 建立“动作语义到连续控制”的稳定接口。

### 2.3 语义对齐的 3D 几何集成

#### 核心思想

VLM 的 2D 视觉 encoder 已经通过大规模视觉语言预训练与语言语义空间对齐，因此不能随意破坏。  
但机器人操作需要 3D 几何信息，因此论文引入一个额外的 3D spatial encoder，并将其与 VLM 表示空间逐步对齐。

---

#### 具体做法

GEAR-VLA 使用：

- **冻结的 2D visual encoder**：来自 Qwen2.5-VL，用于保留原有语义能力；
- **可训练的 3D spatial encoder**：使用 VGGT，用于提取多视角 3D 几何结构；
- **扩展 visual projector**：将 2D 语义特征和 3D 几何特征融合到 LLM token 空间。

融合公式为：

\[
Z_{vis} = [H_{2D}; H_{3D}]W_{vis} + b
\]

其中：

- \(H_{2D}\)：2D visual encoder 特征；
- \(H_{3D}\)：VGGT 3D spatial encoder 特征；
- \(W_{vis}\)：扩展后的视觉投影矩阵。

初始化方式为：

\[
W^{(0)}_{vis} = [W_{Qwen}; 0]
\]

也就是说：

- 2D 部分继承 Qwen2.5-VL 原始 projector；
- 3D 部分零初始化。

这样做的好处是：

- 初始阶段不会扰乱原 VLM 的视觉 token 分布；
- 训练过程中 3D 特征逐渐注入；
- 最终得到同时具备语义和几何结构的视觉表示。

---

### 2.4 本体规范化

#### 核心思想

不同机器人之间的差异不应该混入高层语义策略表示，而应该限制在低层状态输入和动作输出接口中。

因此，GEAR-VLA 采用：

- 本体感知状态输入；
- 本体不变动作空间；
- 每个机器人一个轻量状态 projector；
- 不使用机器人专用语义 soft prompt。

---

#### 动作输出：相对末端执行器动作

对于当前末端位姿 \(T^{ee}_t \in SE(3)\)，未来第 \(i\) 步目标末端位姿为 \(T^{ee}_{t+i}\)。  
GEAR-VLA 将动作表示为相对于当前末端执行器的 SE(3) 变换：

\[
\Delta T_{t+i} = (T^{ee}_t)^{-1} T^{ee}_{t+i}
\]

动作 chunk 为：

\[
A^{rel}_t = [\Delta T_{t+1}, \Delta T_{t+2}, ..., \Delta T_{t+K}, g_{t+1:t+K}]
\]

其中 \(g\) 表示 gripper command。

这种设计相比：

- **绝对动作**：更少依赖机器人基座坐标和工作空间；
- **逐步 delta action**：避免 chunk 内误差累积；
- **机器人专用 action head**：更便于跨本体共享。

---

#### 状态输入：末端位姿 + 关节角

状态表示为：

\[
s_t = \{T^{ee}_t, q_t\}
\]

其中：

- \(T^{ee}_t\)：末端执行器位姿；
- \(q_t\)：关节角。

每个机器人本体 \(e\) 有一个轻量状态 projector：

\[
z_{s_t} = f^s_{\psi_e}(s_t)
\]

当迁移到新机器人时，采用两阶段轻量适配：

1. 冻结共享 VLA backbone，只训练新机器人的 state projector；
2. 进行轻量端到端微调。

## 6. 论文主要结论与发现

论文得到的主要结论包括：

1. **统一几何感知动作表示可以显著提升 VLA 泛化能力**  
   GEAR-VLA 在仿真和真实任务中均取得领先结果，说明语义、几何和动作表示的统一建模对机器人操作有效。

2. **粗到细动作学习优于直接连续轨迹监督**  
   FAST token 与 latent action ID 都带来性能提升，说明离散动作语义和视频动态学习有助于构建更稳定的动作表示。

3. **3D 几何特征需要与 VLM 语义空间对齐**  
   直接注入或错误初始化 3D 特征会显著损害性能；冻结 2D 语义路径、零初始化 3D connector、可训练 VGGT 是关键设计。

4. **本体差异应限制在低层接口，而不是注入共享语义空间**  
   X-VLA-style soft prompt 表现不如 GEAR-VLA 的 embodiment-aware state projector，说明机器人身份信息不应过度进入高层策略表示。

5. **跨本体迁移具有实际可行性**  
   在预训练未见过的 LDT-01 上仍达到 81.0% 成功率，说明该方法具有较强迁移能力。

6. **真实未见物体抓取性能显著提升**  
   在 212 个未见物体、6360 次真实试验中达到 90.1%，尤其在 irregular、tool、dense clutter 场景中优势明显。

## 7. 优点与亮点

### 7.1 方法设计清晰且针对性强

论文将 VLA 泛化问题拆解为：

- 动作表示；
- 3D 空间表示；
- 跨本体表示。

三个模块分别对应三个关键瓶颈，结构清晰。

---

### 7.2 粗到细动作学习较有启发性

先通过 token-level embodied pretraining 学习动作语义，再通过 DiT 动作专家生成连续轨迹，避免了直接用连续控制损失破坏 VLM 表示。

尤其是：

- latent action IDs 利用无动作标签视频；
- FAST tokens 利用机器人轨迹；
- DiT 与 VLM 梯度解耦；

这些设计共同提高了训练稳定性和泛化能力。

---

### 7.3 3D 集成方式较稳健

论文没有简单拼接深度图或直接加入 3D positional encoding，而是：

- 冻结 VLM-aligned 2D encoder；
- 使用 VGGT 提取多视角 3D 结构；
- 通过零初始化 connector 渐进注入；
- 训练 3D spatial encoder 与 VLA 表示对齐。

这比粗暴融合 3D 特征更稳定。

---

### 7.4 本体规范化思路实用

相对末端执行器动作、轻量 state projector 和两阶段适配，使模型能较好迁移到新机器人。  
这种做法比为每个机器人引入专用 action head 或 soft prompt 更利于共享高层表示。

---

### 7.5 实验覆盖广

论文不仅在标准仿真 benchmark 上报告性能，还做了：

- 真实双臂折叠与翻转任务；
- 预训练未见本体迁移；
- 大规模真实未见物体抓取；
- 多项消融；
- 数据效率实验。

实验广度明显强于许多只在仿真或少量真实任务上评估的 VLA 工作。

---

### 7.6 真实抓取 benchmark 规模较大

6360 次真实机器人试验、212 个未见物体和三类复杂场景，使实验结果较有说服力。  
尤其是只使用第一帧 mask，而不依赖持续 mask tracking，也体现了模型的目标保持和空间泛化能力。

---

## 8. 不足与局限

### 8.1 训练资源需求极高

GEAR-VLA 使用：

- 240 张 H200 进行 350K + 700K iterations 的大规模训练；
- 56 张 H200 进行 LIBERO 微调。

这意味着该方法复现门槛极高，普通实验室难以验证或扩展。

---

### 8.2 数据规模优势可能影响公平性

论文使用了大量机器人、人类和 web 视频数据，包括：

- OXE；
- AgiBot；
- Droid；
- RoboMind；
- Galaxea；
- Bridge V2；
- HoloAssist；
- Ego4D；
- EgoDex；
- EgoVid；
- Egocentric-10k 等。

因此，GEAR-VLA 的性能提升可能同时来自方法设计和大规模数据/算力，二者贡献难以完全分离。

---

### 8.3 对 3D backbone 和多视角输入有依赖

GEAR-VLA 使用 VGGT 和多视角图像。  
这在真实部署中可能带来限制：

- 需要较稳定的多摄像头配置；
- 缺失视角虽可 padding，但几何质量可能下降；
- VGGT 特征计算可能增加推理成本；
- 对相机标定、视角覆盖和图像质量可能较敏感。

---

### 8.4 未充分报告推理效率

论文没有详细说明：

- 推理延迟；
- DiT action expert 采样步数或速度；
- 是否满足高频闭环控制；
- 在真实机器人上的实时性指标；
- 3D spatial encoder 带来的额外计算开销。

这些对于实际部署非常重要。

---

### 8.5 真实任务类型仍有限

虽然真实实验规模较大，但任务类型主要集中在：

- 双臂折叠；
- 包裹翻转；
- pick-and-place / grasping。

尚未充分覆盖：

- 长时序多阶段任务；
- 工具使用；
- 高精度装配；
- 柔性物复杂操作；
- 动态环境交互；
- 人机协作场景。

---

### 8.6 跨本体验证仍可进一步扩大

LDT-01 是预训练未见本体，验证价值较高。  
但论文真实跨本体实验主要是两个双臂平台，尚未充分展示在以下机器人上的泛化：

- 单臂工业机械臂；
- 移动操作机器人；
- 灵巧手；
- 人形机器人；
- 不同 gripper 类型；
- 非末端执行器式操作平台。

---

### 8.7 失败案例与安全性分析不足

论文主要报告成功率，但较少分析：

- 失败类型；
- 碰撞风险；
- 错误 grasp 的原因；
- 对遮挡、透明物体、反光物体的鲁棒性；
- 长时间运行稳定性；
- 安全约束和异常恢复机制。

这些是机器人真实部署中的关键问题。

---

### 8.8 mask-guided grasping pipeline 仍依赖外部 SAM

在 universal grasping 中，流程使用 VLM grounding 后调用 SAM 生成第一帧目标 mask。  
这说明系统仍部分依赖外部分割模型，可能受到以下因素影响：

- grounding box 错误；
- SAM 分割错误；
- mask 与目标不一致；
- 多相似物体场景中的实例混淆。

虽然 GEAR-VLA 后续只需第一帧 mask，但前端目标指定的鲁棒性仍是潜在瓶颈。

---

## 总体评价

GEAR-VLA 是一篇系统性较强的 VLA 论文，核心贡献在于将动作语义、3D 几何和跨本体迁移统一到一个几何感知动作表示框架中。它通过粗到细动作学习避免低层轨迹监督直接破坏 VLM 表示，通过语义对齐的 3D 融合增强空间理解，通过本体规范化提升跨机器人迁移能力。

实验结果覆盖仿真、真实双臂任务、未见机器人本体和大规模通用抓取，整体表现非常强。其主要不足是资源需求极高、复现门槛大、推理效率和安全性分析不足，以及真实任务和机器人类型仍有进一步扩展空间。

总体而言，论文对未来通用机器人操作模型具有较高参考价值，尤其适合启发后续关于 **动作表示学习、3D-VLA 融合、跨本体策略共享** 的研究。
