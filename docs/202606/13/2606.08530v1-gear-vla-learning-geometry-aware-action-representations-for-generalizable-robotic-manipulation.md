## 摘要
视觉-语言-动作（Vision-Language-Action, VLA）模型在基准测试中表现强劲，但在面对未见过的物体、背景变化以及不同机器人本体时，仍难以在真实世界部署中取得稳健表现。我们认为，这源于缺乏统一的几何感知操作表征，使现有 VLA 易受低层轨迹监督、错位的 3D 特征以及本体差异的影响。为此，我们提出 GEAR-VLA，这是一种用于学习统一几何感知动作表征的 VLA 框架，面向可泛化的机器人操作。GEAR-VLA 采用由粗到细的动作学习：多源具身预训练首先使 VLM 具备具身推理能力和离散动作理解能力，随后潜在动作 token 将动作语义连接到一个梯度解耦的 DiT 连续动作专家。该框架进一步通过语义对齐的 3D 集成，将可训练的 3D 空间骨干网络与 VLA 表征对齐，同时冻结原本与 VLM 对齐的视觉通路。为在不同机器人之间共享该表征，GEAR-VLA 采用本体规范化，其中本体感知状态和本体不变动作将机器人差异限制在低层接口中。大量仿真与真实世界实验表明其具有很强的泛化能力：GEAR-VLA 在 LIBERO、零样本 LIBERO-Plus 和 RoboTwin 2.0 上达到最先进性能，在 AgileX 上实现 85.9% 的成功率，在预训练中未见过的 LDT-01 本体上实现 81.0% 的成功率，并在包含 212 个未见物体、共 6,360 次试验的通用抓取基准上取得 90.1% 的成功率。代码和模型将发布于 https://github.com/babynabeauty/GEAR-VLA。

## Abstract
Vision-Language-Action (VLA) models achieve strong benchmark performance but still struggle in real-world deployment with unseen objects, background shifts, and different robot embodiments. We argue that this stems from the lack of a unified geometry-aware manipulation representation, leaving existing VLAs vulnerable to low-level trajectory supervision, misaligned 3D features, and embodiment differences. To address this, we propose GEAR-VLA, a VLA framework for learning unified geometry-aware action representations for generalizable robotic manipulation. GEAR-VLA adopts coarse-to-fine action learning, where multi-source embodied pretraining equips the VLM with embodied reasoning and discrete action understanding before latent action tokens connect action semantics to a gradient-decoupled DiT continuous action expert. It further performs semantic-aligned 3D integration by aligning a trainable 3D spatial backbone with the VLA representation while freezing the original VLM-aligned visual pathway. To share this representation across robots, GEAR-VLA uses embodiment canonicalization, where embodiment-aware states and embodiment-invariant actions confine robot differences to the low-level interface. Extensive simulation and real-world experiments demonstrate strong generalization: GEAR-VLA achieves state-of-the-art performance on LIBERO, zero-shot LIBERO-Plus, and RoboTwin 2.0, reaches 85.9% success on AgileX and 81.0% on the pretraining-unseen LDT-01 embodiment, and obtains 90.1% success on a 6,360-trial universal grasping benchmark with 212 unseen objects. Code and models will be released at https://github.com/babynabeauty/GEAR-VLA.

## 2. 方法论：核心思想、关键技术与流程

GEAR-VLA 的整体框架由三部分组成：

1. **Coarse-to-Fine Action Learning：粗到细动作学习**
2. **Semantic-Aligned 3D Integration：语义对齐的 3D 几何集成**
3. **Embodiment Canonicalization：机器人本体规范化**

其总体输入为：

- 多视角 RGB 图像；
- 语言指令；
- 机器人状态；
- 当前末端执行器位姿与关节角。

输出为未来一段时间的动作序列，即 **action chunk**。

---

### 2.1 粗到细动作学习：从离散动作语义到连续控制

#### 核心思想

作者认为，如果直接用连续轨迹监督 VLM，模型容易退化为低层轨迹拟合器。因此，GEAR-VLA 先让 VLM 学会“动作语义”和“具身推理”，再通过独立的连续动作专家生成精细控制。

该过程分为两个阶段：

#### 阶段一：Embodied VLM Pretraining

模型首先在大量具身数据上进行自回归 token 预测训练，包括：

- 通用视觉问答；
- 空间理解；
- 2D / 3D grounding；
- 轨迹推理；
- pointing；
- affordance 理解；
- mask tracking；
- 机器人操作视频；
- 人类操作视频；
- 网页视频等。

对于有机器人动作标签的数据，使用 **FAST-style action tokenization** 将连续动作序列离散化为动作 token。

对于没有动作标签的视频，作者训练了一个 **causal VQ-VAE latent action tokenizer**，从视频中提取 latent action IDs。

这些 latent action IDs 用于学习动作相关的视觉动态，例如：

- 物体移动；
- 接触变化；
- 手或机器人运动；
- 场景状态转移。

预训练目标统一为自回归 token 预测：

\[
L_{VLM}=-\sum_i \log p_\theta(y_i|y_{<i}, O, l, s)
\]

其中：

- \(O\)：多视角观测；
- \(l\)：语言指令；
- \(s\)：机器人状态；
- \(y_i\)：文本、grounding、planning、FAST action token 或 latent action token 等目标 token。

#### 阶段二：Continuous Action Generation

在 VLM 已经获得动作语义之后，作者引入一个基于 DiT 的连续动作专家。

关键点是：

- DiT 不直接访问完整 VLM 表征；
- 它只使用 VLM 产生的 **latent action token 的 K/V cache**；
- 连续动作损失不会反传到 VLM 主干；
- 通过 stop-gradient 实现梯度解耦。

连续控制使用 flow matching 损失：

\[
L_{FM}=\mathbb{E}_{A,\epsilon,\tau}
\left[
\|v_\phi(A_\tau,\tau,sg(h_{la}))-(A-\epsilon)\|_2^2
\right]
\]

其中：

\[
A_\tau=\tau A+(1-\tau)\epsilon
\]

- \(A\)：目标连续动作 chunk；
- \(\epsilon\)：高斯噪声；
- \(\tau\)：flow time；
- \(h_{la}\)：latent action token cache；
- \(sg(\cdot)\)：stop-gradient。

这一设计使得 VLM 负责高层语义与推理，DiT 负责低层连续控制，避免连续动作回归破坏 VLM 的语义空间。

### 2.2 语义对齐的 3D 几何集成

#### 核心思想

机器人操作高度依赖 3D 空间结构，但 VLM 的视觉编码器主要学习 2D 语义。直接注入深度或 3D 特征可能导致特征分布错位。

GEAR-VLA 的做法是：

- 保留原始 2D 视觉通路；
- 冻结已经和 LLM 对齐的 2D visual encoder；
- 新增可训练的 3D spatial encoder；
- 使用 VGGT 作为 3D 空间骨干；
- 通过零初始化的 connector 将 3D 特征逐渐注入 VLM 视觉 token 空间。

#### 具体机制

模型提取两类特征：

- \(H_{2D}\)：2D visual encoder 输出；
- \(H_{3D}\)：VGGT 3D spatial encoder 输出。

二者沿特征维拼接后，通过扩展视觉投影器映射到 LLM token 空间：

\[
Z_{vis}=[H_{2D};H_{3D}]W_{vis}+b
\]

其中投影矩阵初始化为：

\[
W^{(0)}_{vis}=[W_{Qwen};0]
\]

也就是说：

- 2D 分支继承 Qwen2.5-VL 的原始 projector；
- 新增 3D 分支零初始化；
- 初始时不会扰乱原有 VLM 表征；
- 训练中逐步学习如何融合 3D 几何结构。

这一设计的目标是让模型同时利用：

- 2D 语义线索；
- 多视角 3D 结构；
- 物体形状；
- 空间关系；
- 操作几何约束。

### 2.3 机器人本体规范化

#### 核心思想

不同机器人之间存在结构差异。如果使用机器人特定 prompt 或独立 action head，机器人身份可能被注入高层语义空间，从而削弱共享表征的迁移性。

GEAR-VLA 的策略是：

> 将机器人差异限制在低层状态与动作接口中，而不是注入共享 VLA 表征。

#### 动作输出侧：相对末端执行器动作

作者使用 **Relative End-effector Action** 作为统一动作空间。

设当前末端执行器位姿为：

\[
T_t^{ee}
\]

第 \(i\) 个未来目标末端位姿为：

\[
T_{t+i}^{ee}
\]

则相对动作表示为：

\[
\Delta T_{t+i}=(T_t^{ee})^{-1}T_{t+i}^{ee}
\]

动作 chunk 为：

\[
A_t^{rel}=[\Delta T_{t+1},\Delta T_{t+2},...,\Delta T_{t+K},g_{t+1:t+K}]
\]

其中 \(g\) 表示夹爪控制序列。

这一表示方式有三个优点：

- 相比绝对位姿动作，减少对机器人基座坐标和工作空间的依赖；
- 相比逐步 delta action，避免 action chunk 内误差累积；
- 不同机器人可以共享相对末端动作空间。

#### 状态输入侧：本体感知状态投影

模型输入状态包括：

\[
s_t=\{T_t^{ee},q_t\}
\]

其中：

- \(T_t^{ee}\)：末端执行器位姿；
- \(q_t\)：关节角。

不同机器人使用轻量级状态投影器：

\[
z_{s_t}=f_{\psi_e}^{s}(s_t)
\]

其中 \(e\) 表示机器人本体。

迁移到新机器人时采用两阶段适配：

1. 冻结共享 VLA 主干，只训练新机器人的 state projector；
2. 再进行轻量端到端微调。

## 3. 实验设计：数据集、场景、benchmark 与对比方法

本文实验覆盖仿真和真实机器人两大类，规模较大。

---

### 3.1 仿真实验 benchmark

使用了三个主要仿真 benchmark：

#### 1. LIBERO

用于评估标准机器人操作能力，包括：

- Spatial；
- Object；
- Goal；
- Long-horizon tasks。

#### 2. LIBERO-Plus

用于评估零样本鲁棒性和分布外泛化，包括七类扰动：

- Camera；
- Robot；
- Language；
- Light；
- Background；
- Noise；
- Layout。

#### 3. RoboTwin 2.0

大规模双臂机器人操作 benchmark，包含 clean 和 randomized 两种设置，用于评估：

- 多任务操作；
- 双臂控制；
- 随机化环境下的泛化能力。

---

### 3.2 真实机器人实验

#### 1. AgileX 双臂机器人

使用 14-DoF AgileX 双臂机器人，评估三个真实世界双臂任务：

- T-shirt folding；
- Shorts folding；
- Parcel-label flipping。

每个任务收集：

- 200 条双臂遥操作示范；
- 共 600 条训练样本。

训练时只使用一种固定颜色物体，测试时使用三种未见颜色 / 外观 / 尺寸，以评估真实世界外观泛化。

#### 2. LDT-01 跨本体迁移

使用 16-DoF LDT-01 双臂机器人。

重要的是：

- LDT-01 在预训练中没有类似本体；
- 用于测试模型向未见机器人形态的迁移能力。

同样每个任务收集 200 条示范，并在未见外观和尺寸上测试。

#### 3. 通用物体抓取 benchmark

作者构建了一个大规模真实世界 universal grasping benchmark。

训练集：

- 35 个物体；
- 每个物体 100 条示范；
- 总计约 3500 条示范。

测试集：

- 212 个未见物体；
- 物体类别包括：
  - axisymmetric；
  - block-like；
  - irregular；
  - tool；
  - bagged。

测试场景：

- Sparse clutter；
- Dense clutter；
- Background / lighting variation。

每个物体测试 10 次。

总测试量：

- 每个方法每个 setting 2120 次；
- 三个 setting 共 6360 次真实机器人试验。

该任务采用 mask-guided prompt：

- 首帧用 mask 指定目标物体；
- 模型需要在后续观察中持续定位并操作目标；
- 与 DexGraspVLA 不同，GEAR-VLA 不需要持续 mask tracking，只需要首帧 mask。

## 6. 主要结论与发现

### 6.1 仿真实验结论

GEAR-VLA 在主要仿真 benchmark 上达到或超过现有方法。

#### LIBERO

- GEAR-VLA 平均成功率：98.7%；
- 高于 ACoT 的 98.5% 和 X-VLA 的 98.1%。

#### LIBERO-Plus

- GEAR-VLA 零样本平均成功率：88.7%；
- 高于 ACoT 的 86.6% 和 π0.5 的 85.7%。

#### RoboTwin 2.0

- Clean：91.1%；
- Randomized：89.9%；
- 明显高于 ACoT、X-VLA、π0.5 等方法。

这表明 GEAR-VLA 不只是适配标准任务，也在分布变化和随机化场景下具有较强鲁棒性。

---

### 6.2 真实双臂操作结论

#### AgileX

GEAR-VLA 在三个真实双臂任务上平均成功率：

- 85.9%。

相比主要 baseline：

- π0：64.2%；
- X-VLA：83.2%；
- π0.5：80.6%；
- ACoT：77.8%。

#### LDT-01

GEAR-VLA 在预训练未见过的 LDT-01 机器人上平均成功率：

- 81.0%。

相比 π0.5 提升约 7.1 个百分点。

这说明本体规范化接口对跨机器人迁移有效。

---

### 6.3 通用抓取结论

在 212 个未见物体、6360 次真实机器人试验中：

- GEAR-VLA：90.1%；
- DexGraspVLA：84.4%；
- π0.5：79.1%。

GEAR-VLA 在以下类别上尤其有优势：

- irregular objects；
- tool objects；
- dense clutter；
- background / lighting variation。

这说明其几何感知表示对复杂物体形状和干扰场景有明显帮助。

---

### 6.4 消融实验结论

消融结果支持作者的核心设计：

- 去掉 latent action IDs：平均下降 1.6 点；
- 去掉 FAST tokens：下降 2.5 点；
- 去掉 VGGT：下降 3.6 点；
- 冻结 VGGT：下降 3.5 点；
- 去掉 zero-init 3D projector：下降 6.8 点；
- 解冻原始 2D ViT：下降 2.1 点；
- one-stage adaptation：下降 9.7 点；
- 使用 X-VLA soft prompt：下降 3.7 点；
- 去掉 embodiment-specific state projector：下降 2.0 点。

尤其值得注意的是，zero-initialized 3D connector 和两阶段本体适配对稳定训练和跨本体迁移非常关键。

## 7. 优点与亮点

### 7.1 方法设计完整，针对性强

GEAR-VLA 对现有 VLA 的三个关键痛点分别提出了解决方案：

- 动作表示问题：粗到细动作学习；
- 3D 几何错位问题：语义对齐 3D 集成；
- 跨机器人迁移问题：本体规范化。

这三个模块逻辑上相互配合，而不是孤立堆叠。

### 7.2 高层语义与低层控制解耦

通过 latent action token cache 连接 VLM 和 DiT action expert，并使用 stop-gradient，避免连续动作损失破坏 VLM 表征。

这一设计较好地区分了：

- VLM 的语义推理能力；
- DiT 的连续轨迹生成能力。

### 7.3 3D 几何集成方式较稳健

冻结 2D ViT、零初始化 3D connector、训练 VGGT 适配 VLA 空间，是较稳健的工程设计。

它避免了直接注入 3D 特征导致的语义空间扰动。

### 7.4 跨本体设计较有实际价值

相对末端执行器动作和本体特定状态投影器，使机器人差异主要留在低层接口，而不是污染共享语义空间。

这对于真实多机器人部署较有意义。

### 7.5 实验规模大

论文不仅做了标准仿真 benchmark，还包括：

- 真实双臂操作；
- 未见机器人本体迁移；
- 212 个未见物体抓取；
- 6360 次真实机器人试验；
- 多组系统性消融。

实验覆盖面优于许多仅在仿真或少量真实任务上验证的 VLA 工作。

### 7.6 利用无动作标签视频

通过 causal VQ-VAE 从人类视频和网络视频中提取 latent action IDs，使模型可以利用大规模无动作标注视频学习操作动态。

这对于扩展 VLA 数据来源很有价值。

---

## 8. 不足与局限

### 8.1 训练资源需求极高

GEAR-VLA 需要：

- 240 张 H200 GPU；
- 350K + 700K iterations 的大规模训练；
- 多源大规模数据；
- 复杂的两阶段训练与下游微调。

这使得方法复现成本非常高，普通实验室难以验证。

### 8.2 数据依赖强

模型使用大量开源和自建数据，包括：

- 机器人数据；
- 人类操作视频；
- 自建空间理解数据；
- 自建 pointing / affordance / trajectory / mask tracking 数据。

自建数据的质量、分布和标注流程会显著影响模型性能，但论文中对数据构建细节的透明度仍有限。

### 8.3 benchmark 可能存在自建偏差

真实 universal grasping benchmark 是作者自建的，虽然规模很大，但仍可能存在以下风险：

- 任务设计更适合 GEAR-VLA 的 mask-guided 能力；
- 物体类别划分和训练物体选择可能影响泛化结论；
- 背景、光照、桌面布局虽然随机，但仍未必覆盖更开放的家庭或工业环境。

需要第三方独立复现来验证通用性。

### 8.4 与 baseline 的资源公平性不完全清楚

虽然论文比较了大量强方法，但不同方法的：

- 预训练数据规模；
- 模型规模；
- 训练算力；
- 下游调参；
- 真实机器人适配流程；

可能并不完全一致。因此，性能提升既可能来自方法设计，也可能部分来自更大规模训练和数据。

### 8.5 仍依赖任务级微调

在真实 AgileX 和 LDT-01 双臂任务中，每个任务仍需要 200 条示范进行适配。

这说明模型虽然泛化能力较强，但并非完全 zero-shot 操作系统。

### 8.6 首帧 mask-guided 抓取仍需额外目标指定机制

通用抓取中虽然只需要首帧 mask，但流程仍涉及：

- 语言 grounding；
- SAM 生成 mask；
- mask-guided prompt 重写；
- 当前多视角观测输入。

这在实际部署中增加了系统复杂度，也可能受到 grounding 或 SAM 错误影响。

### 8.7 长时序任务能力仍不明确

LIBERO Long 和部分真实操作展示了一定长程能力，但论文主要关注 action chunk 级别的短期控制，动作预测窗口为 1 秒。

对于更长时序、多阶段、需要失败恢复和主动探索的任务，能力仍需进一步验证。

### 8.8 安全性和闭环恢复讨论不足

真实机器人部署中常见问题包括：

- 抓取失败后的恢复；
- 误碰撞；
- 物体滑落；
- 人机共处安全；
- 传感器遮挡；
- 执行器误差。

论文主要报告成功率，对失败模式、安全约束和在线恢复机制讨论较少。

---

## 总体评价

GEAR-VLA 是一篇面向通用机器人操作表征的重要 VLA 工作。它的核心贡献在于将 **动作语义学习、3D 几何对齐和跨机器人本体规范化** 统一到一个框架中，并通过大规模仿真与真实实验验证其泛化能力。

从实验结果看，GEAR-VLA 在 LIBERO、LIBERO-Plus、RoboTwin 2.0、真实双臂操作和通用抓取中均表现突出，尤其在复杂几何、未见物体和跨本体迁移方面具有明显优势。

不过，该方法高度依赖大规模数据与昂贵算力，复现门槛高；同时，真实 benchmark 和部分数据为自建，仍需要更开放、独立的第三方验证。总体而言，论文提出的表征学习思路具有较强启发性，对后续 VLA、机器人基础模型和跨本体操作策略设计具有重要参考价值。
