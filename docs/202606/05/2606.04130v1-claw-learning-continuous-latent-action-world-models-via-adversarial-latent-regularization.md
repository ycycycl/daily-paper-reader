## 摘要
我们提出了 CLAW，一个完全端到端的自监督框架，用于直接从无动作标注视频中联合学习世界模型与连续潜在动作表征。该方法利用对抗式潜在正则化和基于扩散的视频生成，在不依赖任何动作标签或注释的情况下，捕捉结构化且具有语义意义的动作表征，同时建模丰富且具预测性的环境动态。通过同时训练潜在动作模型和世界模型，CLAW 学会仅从视觉观测中推理所推断动作如何诱导环境转移。我们表明，所得的潜在动作世界模型同时支持从观察中进行模仿学习和目标导向规划。在模仿学习中，从原始视频中提取的潜在动作能够实现行为克隆。对于规划，CLAW 生成潜在动作序列，并将其映射为可执行动作以达到期望目标。在多样化任务和具身形态上的大量实验表明，CLAW 能够产生具有语义意义的潜在动作表征，支持有效的动作迁移，并实现从观察中进行规划和模仿，性能优于现有方法。

## Abstract
We introduce CLAW, a fully end-to-end self-supervised framework for learning a world model jointly with continuous latent action representations directly from action-free videos. Our approach leverages adversarial latent regularization and diffusion-based video generation to capture structured and semantically meaningful action representations while modeling rich, predictive environment dynamics, without relying on any action labels or annotations. By simultaneously training the Latent Action Model and world model, CLAW learns to reason about how inferred actions induce environment transitions from visual observations alone. We show that the resulting latent action world model supports both imitation learning from observation and goal-directed planning. In imitation learning, latent actions extracted from raw videos enable behavior cloning. For planning, CLAW generates sequences of latent actions and maps them to executable actions to reach desired goals. Extensive experiments across diverse tasks and embodiments demonstrate that CLAW produces semantically meaningful latent action representations, supports effective action transfer, and enables planning and imitation from observation, outperforming existing methods.

### 1.1 论文关注的问题

论文 **CLAW: Learning Continuous Latent Action World Models via Adversarial Latent Regularization** 研究的是：

> 如何仅从“没有动作标签的视频”中，端到端学习一个既能预测环境动态、又能推断连续潜在动作的世界模型。

传统机器人世界模型通常学习如下形式的动力学：

\[
P(o_{t+1} \mid o_t, a_t)
\]

即给定当前观测 \(o_t\) 和真实动作 \(a_t\)，预测下一帧或未来状态。但这依赖动作标签，而真实世界中大量可用数据是视频，例如人类操作物体、机器人自由探索、游戏录像等，这些数据通常没有可执行动作标注。

因此，本文希望从无动作视频中学习：

\[
P(o_{t+1} \mid o_t, z_t)
\]

其中 \(z_t\) 不是人工给定动作，而是模型从相邻视觉帧中推断出的 **潜在动作 latent action**。

### 1.2 背景与挑战

这类“潜在动作世界模型”有重要意义：

- 可以利用大规模无标注视频数据；
- 可以支持从观察中模仿学习，即 **Imitation Learning from Observation, ILfO**；
- 可以支持视觉目标导向规划；
- 有潜力跨任务、跨实体迁移，因为潜在动作不依赖某一机器人的具体控制接口。

但核心难点是：  
学习到的潜在动作很容易出现两种退化：

1. **未来信息泄漏 future information leakage**
   - LAM 可能直接把下一帧 \(o_{t+1}\) 的视觉内容编码进 \(z_t\)；
   - 这样世界模型只是在“解码未来帧”，而不是理解动作语义。

2. **表示坍塌 representation collapse**
   - 潜在动作可能变得无信息或不可控；
   - 不能用于规划或动作迁移。

已有方法常用离散化、VQ-VAE、β-VAE、瓶颈约束或少量动作监督来缓解问题，但这些方法存在动作粒度有限、需要标注或训练阶段割裂等问题。

### 1.3 论文整体含义

CLAW 的核心贡献是提出一个 **端到端、自监督、连续潜在动作世界模型学习框架**，直接从无动作视频中联合训练：

- 潜在动作模型 LAM；
- 扩散式视频世界模型；
- 对抗式潜在正则化机制。

其目标是让潜在动作真正表示“导致状态变化的动作因素”，而不是携带未来图像内容。

## 2. 方法论：核心思想、关键技术与训练流程

## 2.1 整体框架

CLAW 由两个主要模块组成：

1. **Latent Action Model, LAM**
   - 输入相邻两帧 \((o_t, o_{t+1})\)；
   - 输出连续潜在动作 \(z_t\)。

2. **Diffusion-based World Model**
   - 输入初始帧 \(o_t\) 和一段潜在动作序列 \(\{z_t, z_{t+1}, ...\}\)；
   - 预测未来视频帧。

整体关系为：

\[
z_t = \text{LAM}_\theta(o_t, o_{t+1})
\]

\[
\hat{o}_{t+1:t+c} = \text{WorldModel}_\phi(o_t, z_{t:t+c})
\]

LAM 和世界模型不是分开训练，而是端到端联合训练。

---

## 2.2 潜在动作模型 LAM

LAM 使用 ViT 结构处理连续两帧图像。

### 输入

\[
(o_t, o_{t+1})
\]

### 输出

\[
z_t \in \mathbb{R}^n
\]

文中所有实验中潜在动作维度为：

\[
n = 32
\]

### 技术细节

- 图像被切分为 patch；
- 加入空间位置编码和时间位置编码；
- 使用一个可学习的 action token 汇聚两帧之间的转移信息；
- 最后通过 MLP 投影为潜在动作向量；
- 对输出做 \(\ell_2\) 归一化，使其位于单位球面上。

即：

\[
z_t = \frac{\text{MLP}(\text{ViT}(o_t, o_{t+1}))}{\|\text{MLP}(\text{ViT}(o_t, o_{t+1}))\|_2}
\]

这样做的目的是让潜在动作空间更紧凑、可比较，并方便最近邻检索。

---

## 2.3 扩散式世界模型

CLAW 的世界模型采用基于 U-ViT 的扩散视频生成模型。

### 建模目标

给定当前帧和潜在动作序列，预测未来帧：

\[
P(o_{t+1:t+c} \mid o_t, z_{t:t+c})
\]

### 扩散训练目标

论文采用 variance-preserving SDE 形式的扩散过程。对于干净图像 \(x_0\)，加噪过程为：

\[
x_t = \sqrt{\alpha(t)} x_0 + \sqrt{1-\alpha(t)} \epsilon
\]

其中：

\[
\epsilon \sim \mathcal{N}(0, I)
\]

模型学习预测噪声：

\[
\mathcal{L} =
\mathbb{E}_{t,\epsilon}
\left[
\|\epsilon - s_\phi(x_t, t, z_t, o_i)\|^2
\right]
\]

其中 \(s_\phi\) 是扩散模型的 score/noise prediction 网络。

### 输入 token 结构

世界模型输入包括：

- 扩散时间步 embedding；
- 潜在动作 token；
- 初始条件帧 \(o_i\) 的 patch；
- 被加噪的目标帧 patch。

### 因果注意力 mask

论文设计了一个结构化 causal attention mask，以防止世界模型在训练时看到未来信息。

具体约束为：

- 当前目标帧 token 只能看：
  - 同一帧内 token；
  - 初始条件帧 token；
  - 当前时间之前或当前对应的动作 token；
- 不能看：
  - 未来帧；
  - 未来动作 token。

这保证模型学习的是合法的动作条件动力学，而不是利用未来上下文作弊。

---

## 2.4 对抗式潜在正则化

这是 CLAW 的关键创新。

### 问题

如果只用重建未来帧训练 LAM 和世界模型，LAM 很可能把 \(o_{t+1}\) 的视觉细节直接编码进 \(z_t\)，导致信息泄漏。

### 核心想法

如果 \(z_t\) 真的是“动作”，那么仅凭 \(z_t\) 不应该能准确预测下一帧；因为没有当前场景 \(o_t\)，动作本身不足以决定未来图像。

因此，CLAW 同时训练两条路径：

1. **正常路径**
   - 输入：当前帧 \(o_t\) + 潜在动作 \(z_t\)；
   - 目标：准确预测未来帧；
   - 作用：鼓励 \(z_t\) 表示有用动作。

2. **对抗路径**
   - 输入：仅潜在动作 \(z_t\)，不给当前帧；
   - 目标：通过梯度反转层阻止 \(z_t\) 携带未来视觉内容；
   - 作用：如果仅靠 \(z_t\) 就能预测未来，说明 \(z_t\) 泄漏了未来信息，因此要惩罚 LAM。

### 梯度反转机制

在对抗路径中使用 Gradient Reversal Layer：

\[
\frac{\partial L}{\partial z}
\leftarrow
-\alpha
\frac{\partial L}{\partial z}
\]

也就是说，世界模型仍然尝试预测，但传回 LAM 的梯度被反转，使 LAM 被迫去掉能让“仅凭 \(z_t\)”预测未来图像的信息。

### 对抗权重调度

对抗强度 \(\delta\) 随训练逐渐增加：

\[
\delta =
\min
\left(
\frac{2}{1+\exp(-\gamma p)} - 1,
0.999
\right)
\]

其中：

\[
p = \min \left(1.0, \frac{\text{step}}{\text{nsteps}} \right)
\]

论文中设置：

\[
\gamma = 10
\]

这意味着训练前期先让模型学会基本预测，后期逐渐加强防泄漏约束。

---

## 2.5 下游使用方式

CLAW 训练完成后可用于两类任务。

### 2.5.1 从观察中模仿学习 ILfO

流程：

1. 用 LAM 从专家视频中提取潜在动作 \(z_t\)；
2. 训练策略网络：

\[
\pi(z_t \mid o_t)
\]

3. 测试时策略预测潜在动作；
4. 用少量带真实动作标注的参考视频建立 latent-to-action 最近邻映射；
5. 执行最近邻对应的真实动作。

### 2.5.2 视觉目标规划

使用 MPC-CEM 在潜在动作空间中搜索动作序列：

1. 给定当前图像 \(o_t\) 和目标图像 \(o_g\)；
2. 采样多个潜在动作序列；
3. 用世界模型 rollout 预测未来；
4. 根据终端图像与目标图像距离打分；
5. 选择最优序列；
6. 执行前几个动作并重新规划。

规划目标为：

\[
J(z_{1:H}) = d(\hat{o}_{t+H}, o_g)
\]

其中 \(d\) 通常为像素 MSE 或 benchmark 指定的视觉代价。

# 6. 主要结论与发现

## 6.1 CLAW 学到的 latent action 更具语义结构

最近邻检索显示：

- CLAW 检索到的邻居往往具有相似的末端执行器运动或行为语义；
- 不太受颜色、纹理、物体外观等非动作因素影响；
- AdaWorld 和 LAPO 更容易混淆动作和场景外观。

这说明 CLAW 的对抗式正则化有助于减少视觉内容泄漏。

---

## 6.2 CLAW 支持从观察中模仿学习

在 Crafter 中，CLAW 明显优于 baseline：

- Wood Harvest：
  - Random：25.0%；
  - LAPO：35.5%；
  - AdaWorld：43.5%；
  - CLAW：86.0%。

- Avg. Wood：
  - Random：0.28；
  - LAPO：1.12；
  - AdaWorld：1.60；
  - CLAW：4.46。

在 OGBench 中，CLAW 也在 Open Drawer 上达到较高表现，但 Push Button 任务并非全面领先。

---

## 6.3 CLAW 支持视觉规划

在 VP2 中，CLAW 在 6 个任务中的 4 个取得最好或接近最好表现。

例如：

- Open Slide：
  - LAPO：11.67%；
  - AdaWorld：5.83%；
  - CLAW：23.33%。

- Green Button：
  - LAPO：3.33%；
  - AdaWorld：10.00%；
  - CLAW：25.56%。

- Upright Block：
  - LAPO：15.56%；
  - AdaWorld：5.00%；
  - CLAW：43.33%。

但在 Blue Button 和部分 Robosuite Push 任务中，CLAW 不一定领先。

在 Procgen 和 Crafter 视觉规划中，CLAW 在 4 个任务中有 3 个表现最好，平均效果较强。

---

## 6.4 CLAW 可用于真实机器人 latent action pretraining

真实 UR5 任务中，与 Scratch、LAPA、AdaWorld 对比：

- Push：
  - Scratch：0.5；
  - LAPA：0.7；
  - AdaWorld：0.6；
  - CLAW：0.6。

- Cover：
  - Scratch：0.4；
  - LAPA：0.5；
  - AdaWorld：0.6；
  - CLAW：0.7。

- Push-Cover：
  - Scratch：0.4；
  - LAPA：1.3；
  - AdaWorld：1.5；
  - CLAW：1.7。

CLAW 在长程组合任务 Push-Cover 上表现最好，说明其 latent action 对组合行为建模有帮助。

---

## 6.5 对抗正则化提升可控性

在 controllability 实验中，CLAW 相比去掉 adversarial regularization 的版本，在多个环境中有更高 PSNR 和 \(\Delta\)PSNR，说明：

- 预测更准确；
- 随机 latent action 会导致更明显不同的 rollout；
- 世界模型更真正依赖动作条件。

# 7. 优点与亮点

## 7.1 方法层面

- **端到端联合训练**
  - 不像一些方法先训练 LAM 再冻结训练世界模型；
  - LAM 和世界模型可以互相促进。

- **连续 latent action**
  - 相比离散 VQ latent，连续动作更适合细粒度控制；
  - 对机器人控制和视觉规划更自然。

- **对抗式潜在正则化设计巧妙**
  - 通过“仅 latent action 不能预测未来”的原则抑制信息泄漏；
  - 不依赖动作标签；
  - 不依赖硬量化瓶颈。

- **扩散式世界模型**
  - 能生成视觉质量较高的未来帧；
  - 适合复杂视觉环境中的预测与规划。

- **结构化因果注意力 mask**
  - 从 token 级别防止未来信息泄漏；
  - 对视频预测和动作迁移很重要。

---

## 7.2 实验层面

- **覆盖环境多样**
  - 包括机器人模拟、2D 游戏、程序生成环境和真实机器人。

- **评估维度全面**
  - 不仅看预测误差，还看规划、模仿、动作迁移、动作检索和可控性。

- **真实机器人验证**
  - 使用人类视频预训练再迁移到 UR5；
  - 证明方法不仅限于模拟环境。

- **定性分析有说服力**
  - 最近邻检索和 action swap 可视化清楚展示 latent action 的语义结构。

---

# 8. 不足与局限

## 8.1 模型记忆不足

论文自己指出，当前世界模型主要根据最近帧和 latent action 生成未来，没有显式记忆历史生成内容。这在部分可观测环境中可能造成：

- 长程一致性差；
- 隐藏状态丢失；
- 多步 rollout 误差累积。

引入 recurrent memory、latent state memory 或 transformer history buffer 可能是后续方向。

---

## 8.2 仍需要少量动作标注数据

虽然训练 latent action world model 不需要动作标签，但真正执行动作时仍需：

- 少量带真实动作标注的 reference videos；
- 用最近邻把 latent action 映射到可执行动作。

因此，CLAW 并不是完全零动作标注的可执行控制系统。它解决的是大规模无标注视频预训练问题，而非完全免标注部署。

---

## 8.3 扩散模型推理成本可能较高

CLAW 的世界模型使用扩散生成，并且 rollout 时采用 50 个 denoising steps。对于 MPC-CEM 规划，每轮需要评估多个候选动作序列，因此实际计算开销可能较大。

论文没有报告：

- 规划实时性；
- 每次 MPC 的耗时；
- diffusion rollout 的速度；
- 是否适合高频控制。

这对真实机器人在线规划应用是重要限制。

---

## 8.4 消融实验不够系统

虽然论文展示了 No Adv. 消融，但整体消融还可以更完整。例如缺少：

- latent 维度消融；
- action token 数量消融；
- 对抗权重 \(\delta\) 调度消融；
- 是否使用 \(\ell_2\) 归一化的消融；
- diffusion 模型与 deterministic video predictor 的比较；
- reference set 大小对 latent-to-action 映射性能的影响。

---

## 8.5 真实机器人实验规模有限

真实机器人实验只有 3 个任务，每个任务 10 次 trial，且场景集中在桌面操作中的鸭子和杯子任务。虽然有价值，但仍不足以证明方法在复杂真实机器人任务中的泛化能力。

潜在问题包括：

- 物体种类有限；
- 场景变化有限；
- 任务复杂度有限；
- 长程任务只有一个；
- 成功评估依赖 progress score，可能存在主观设计偏差。

---

## 8.6 与 baseline 的公平性仍有讨论空间

不同 baseline 使用的 latent action 类型、训练流程和模型结构不同：

- LAPO 使用离散 latent；
- AdaWorld 使用两阶段 β-VAE；
- CLAW 使用扩散式联合训练和对抗正则。

因此，性能提升可能同时来自：

- 连续 latent action；
- 扩散模型；
- 联合训练；
- 对抗正则；
- 模型容量差异。

论文没有完全拆解这些因素。

---

## 8.7 对复杂真实视频的鲁棒性仍需验证

虽然使用了 Something-Something V2，但主要真实机器人评估仍较窄。真实互联网视频存在：

- 镜头运动；
- 多主体交互；
- 遮挡；
- 非因果剪辑；
- 动作边界不清晰；
- 人类动作与机器人动作形态差异大。

CLAW 在这些情况下的 latent action 是否仍能保持可执行语义，仍需要更大规模验证。

---

# 总结

CLAW 提出了一种从无动作视频中端到端学习连续潜在动作世界模型的方法。其关键在于用 ViT LAM 从相邻帧推断 latent action，用 U-ViT 扩散模型预测未来，并通过对抗式潜在正则化防止 latent action 泄漏未来视觉内容。实验显示 CLAW 在从观察中模仿、视觉规划、动作迁移、真实机器人 latent action pretraining 等任务上整体优于 LAPO、AdaWorld、LAPA 等方法。

这篇论文的主要价值在于证明：  
**仅凭无动作视频，也可以学习具有一定语义结构和可控性的连续潜在动作，并将其用于规划与模仿。**

不过，方法仍依赖少量动作标注进行执行映射，扩散模型计算成本较高，真实机器人实验规模有限，且消融尚不足以完全拆解各设计组件的贡献。
