## 摘要
视觉-语言-动作（Vision-Language-Action, VLA）模型利用预训练视觉-语言模型（Vision-Language Models, VLMs）丰富的世界知识，实现遵循指令的机器人操作。然而，VLM 语义空间与具身控制策略之间的结构性不匹配，往往阻碍精确感知—动作映射的学习。为应对这一挑战，我们提出了 AffordanceVLA，这是一个统一框架，引入结构化可供性预测作为面向任务的中间表示，以建立更精确且鲁棒的感知—动作映射。具体而言，我们通过三个互补组件渐进式地建模操作先验：1）Which2Act，通过视觉潜变量预测实现以物体为中心的定位，以抑制干扰；2）Where2Act，通过可供性图估计实现二维交互定位；3）How2Act，通过三维几何推理指导操作策略。这些可供性线索提供了具有空间锚定、语义条件化且与动作耦合的中间表示，从而自然地桥接视觉、语言与动作。我们将这些模块集成到具有专门化专家的混合 Transformer（Mixture-of-Transformer, MoT）架构中，并采用包含渐进式数据课程的三阶段训练策略来训练模型。为克服机器人数据集中密集可供性标签稀缺的问题，我们还开发了一条鲁棒的自动化数据增强流水线。在仿真和真实世界中的大量实验表明，AffordanceVLA 在多样化操作场景中取得了强劲性能。

## Abstract
Vision-Language-Action (VLA) models leverage the rich world knowledge of pretrained vision-language models (VLMs) to enable instruction-following robotic manipulation. However, the structural mismatch between VLM semantic spaces and embodied control policies often hinders the learning of precise perception--action mappings. To address this challenge, we propose \textbf{AffordanceVLA}, a unified framework that introduces structured affordance forecasting as a task-oriented intermediate representation to establish a more precise and robust perception--action mapping. Specifically, we progressively model manipulation priors through three complementary components: 1) \textbf{Which2Act} for object-centric grounding via visual latent prediction to suppress distractions; 2) \textbf{Where2Act} for 2D interaction localization via affordance map estimation; and 3) \textbf{How2Act} for 3D geometric reasoning to guide manipulation policies. These affordance cues provide spatially grounded, semantically conditioned, and action-coupled intermediate representations, thereby naturally bridging vision, language and action. We integrate these modules into a Mixture-of-Transformer (MoT) architecture with specialized experts and train the model using a three-stage training strategy with a progressive data curriculum. To overcome the scarcity of dense affordance labels in robotic datasets, we also develop a robust automated data augmentation pipeline. Extensive experiments on simulation and real-world demonstrate that AffordanceVLA achieves strong performance across diverse manipulation scenarios.

- 应该以怎样的三维姿态和几何关系去操作？
- 如何将这些信息转化成连续、稳定、物理可执行的动作？

如果仅依靠动作监督直接微调 VLM，模型可能出现：

- 对任务相关空间结构理解不足；
- 被背景、无关物体干扰；
- 长尾场景泛化能力较弱；
- 低层动作损失破坏 VLM 原有的视觉语言能力；

## 2. 方法论：核心思想、关键技术与流程

## 2.1 总体框架

AffordanceVLA 是一个统一的 **感知—预测—动作** 框架，其核心由三个专家模块组成：

1. **Understanding Expert**
   - 负责视觉语言理解；
   - 将图像观测和语言指令融合成指令感知表示。

2. **Affordance Generation Expert**
   - 负责预测结构化可供性信息；
   - 将语义理解转化为可操作的空间与几何先验。

3. **Action Expert**
   - 负责生成机器人动作；
   - 在语义表示和可供性表示的条件下生成连续动作序列。

模型采用 **Mixture-of-Transformer（MoT）架构**，不同专家负责不同阶段的信息处理，避免把理解、可供性预测和动作生成全部混在同一个表示空间中。

---

## 2.2 三类结构化可供性表示

论文将可供性预测分解为三个互补子任务：

### 2.2.1 Which2Act：操作哪个物体

**目标：** 进行物体级定位，找出当前指令下真正需要操作的目标物体。

方法上，模型根据目标 bounding box 裁剪图像区域，并使用冻结的视觉编码器，例如 Flux VAE，提取目标物体的连续视觉 latent。

模型预测该 latent，并使用 MSE 损失进行监督：

- 如果模型能重建目标物体 latent，说明它学会了将语言意图绑定到正确的视觉实体；
- 该设计有助于抑制背景干扰和无关物体。

### 2.2.2 Where2Act：在二维图像上哪里交互

**目标：** 预测二维可供性热力图，指出应当接触或交互的位置。

例如：

- 抓杯子时热力图应集中在杯身或把手；
- 关抽屉时热力图应集中在抽屉把手；
- 按按钮时热力图应集中在按钮区域。

方法上，模型将一维 affordance query token 展开为二维空间分布，使用 Transformer decoder 结合位置编码生成 affordance map。

监督损失为像素级 Binary Cross-Entropy：

- 预测图与 ground-truth heatmap 对齐；
- 使模型具备精细交互区域定位能力。

### 2.2.3 How2Act：如何在三维空间中操作

**目标：** 建模三维形状和空间布局，为低层控制提供几何先验。

How2Act 包含两个分支：

1. **3D Shape Generation**
   - 使用条件扩散模型生成目标物体的三维形状 latent；
   - 通过标准噪声预测目标训练。

2. **Spatial Layout Regression**
   - 回归 10-DoF 空间布局向量；
   - 包括旋转、尺度和平移；
   - 使用 Smooth-L1 损失。

该模块旨在帮助机器人理解目标物体的三维几何结构和空间姿态，尤其对 6-DoF 操作和复杂真实场景有价值。

## 2.3 UAA 渐进式注意力机制

AffordanceVLA 使用 **Understanding–Affordance–Action（UAA）progressive attention**。

其核心原则是：

- 每个专家内部使用双向注意力，保证充分融合；
- 专家之间使用单向因果注意力：
  - Affordance Generation Expert 只能访问 Understanding Expert；
  - Action Expert 可以访问 Understanding 和 Affordance Generation；
  - 反向信息流被禁止。

这样做的目的包括：

- 防止动作信息泄漏到 affordance 预测阶段；
- 保持 affordance 表示的纯净性；
- 避免动作损失破坏视觉语言表示；
- 让动作模块专注于控制，而不是重新学习复杂视觉推理。

## 2.5 自动化可供性标注流水线

由于机器人轨迹数据通常缺少密集 affordance 标签，论文设计了自动化标注流程。

大致流程如下：

1. **关键帧提取**
   - 根据动作序列、夹爪状态变化、速度变化等规则提取关键帧。

2. **指令分解**
   - 使用 Claude Opus 4.5 将长指令分解成每个关键帧对应的原子子指令。

3. **VLM 生成 grounding 查询**
   - 使用 Qwen3-VL 根据图像和子指令生成：
     - detection category；
     - affordance instruction。

4. **视觉 grounding 与分割**
   - 使用 fine-tuned RexOmni 生成 bbox 和 affordance point；
   - 使用 SAM 生成 mask；
   - 使用 SAM-3D 生成 3D shape token 和 layout token。

5. **质量验证**
   - 检查 affordance point 是否落在 bbox 内；
   - 进行 100 轮人工抽检，每轮 30 个样本，总计 3000 个样本；
   - 只有通过质量检查的数据才用于训练。

## 6. 主要结论与发现

### 6.1 仿真 benchmark 结果

在 LIBERO 上，AffordanceVLA full 达到：

- Spatial：98.6；
- Object：98.4；
- Goal：96.2；
- Long：89.8；
- Average：95.8。

该结果略高于 F1-VLA 的 95.7，也高于 Pi0 的 94.4。

在 CALVIN ABC→D 上，AffordanceVLA full 达到：

- 1/5：96.8；
- 2/5：92.0；
- 3/5：87.5；
- 4/5：80.8；
- 5/5：75.9；
- 平均长度：4.33。

该结果优于表中其他方法，包括 Seer-Large 的 4.28。

### 6.2 Stage II 的重要性

AffordanceVLA w/o Stage II 的表现显著低于 full model：

- LIBERO 平均从 86.2 提升到 95.8；
- CALVIN 平均链长度从 3.81 提升到 4.33。

说明大规模 affordance-augmented robotic co-training 是连接通用视觉语言理解与机器人控制的重要桥梁。

### 6.3 三种 affordance 表示均有贡献

去掉任何一个模块都会造成性能下降：

- 去掉 Which2Act：目标物体 grounding 变弱；
- 去掉 Where2Act：精细接触点定位能力下降；
- 去掉 How2Act：三维几何推理能力下降，尤其影响复杂 6-DoF 操作。

但单个模块移除不会导致灾难性崩溃，说明三者不是脆弱串联管线，而是共享表示下的协同机制。

### 6.4 真实机器人结果

Basic tasks：

- Pi0 平均成功率：70.8%；
- AffordanceVLA：88.3%。

Complex tasks：

- Pi0 平均成功率：44.8%；
- AffordanceVLA：82.9%。

在 Drawer 和 Toaster 等相同视觉输入但不同语言指令的任务中，AffordanceVLA 明显更能根据语言选择正确 affordance 区域，说明其指令敏感性更强。

### 6.5 长时序能力

在 Pick all the rubbish 任务中，AffordanceVLA 通过动态重新预测可供性子目标，表现出一定长时序能力：

- 第三次连续拾取成功率显著高于 Pi0；
- 空抓次数明显更少。

但论文也承认，模型本身缺乏显式长期记忆或规划模块，极长时序任务仍是未来方向。

## 7. 优点与亮点

### 7.1 方法设计清晰且针对性强

论文没有简单依赖更大模型或更多数据，而是针对 VLA 的核心结构问题提出中间表示：

- Which2Act 解决“操作哪个物体”；
- Where2Act 解决“在哪里交互”；
- How2Act 解决“如何在三维空间中操作”。

这种分解与机器人操作的实际需求高度一致。

### 7.2 Affordance 作为中间表示较有解释性

相比视频预测等 dense world model 方法，affordance 更紧凑、更任务相关：

- 不需要预测完整未来画面；
- 减少冗余视觉信息；
- 能显式可视化；
- 更便于分析失败原因。

### 7.3 MoT 架构有助于防止表示混淆

通过 Understanding、Affordance Generation、Action 三个专家解耦，模型避免将语义理解、空间预测和低层动作控制完全压进一个单一表示。

UAA 注意力机制也使信息流更加合理。

### 7.4 实验设计较完整

论文不仅报告主结果，还设计了多个关键控制实验：

- data-only control；
- frozen-affordance control；
- same-density control；
- 去除各 affordance 子模块；
- 数据效率曲线；
- 真实机器人语言敏感性实验。

这些实验较好地支撑了论文主张。

### 7.5 数据标注 pipeline 工程价值较高

自动化生成 affordance 标签的 pipeline 是一项重要工程贡献，尤其适用于现有机器人轨迹数据缺少密集空间标签的问题。

---

## 8. 不足与局限

### 8.1 对强外部模型和复杂 pipeline 依赖较大

自动标注流程依赖多个强模型和工具：

- Claude Opus 4.5；
- Qwen3-VL；
- RexOmni；
- SAM；
- SAM-3D；
- Flux VAE。

这会带来：

- 复现难度；
- 成本问题；
- 外部模型偏差传递；
- pipeline 维护复杂度。

### 8.2 训练资源需求较高

训练使用 16 张 NVIDIA H200 GPU，属于非常高的算力配置。

虽然论文报告了训练 steps 和 batch size，但没有报告总训练时间，因此难以完整评估实际训练成本。

### 8.3 真实机器人对比方法有限

真实机器人实验主要与 Pi0 对比，没有与更多强 VLA 方法在同一真实平台上对比，例如：

- OpenVLA；
- gr00t-N1；
- F1-VLA；
- SpatialVLA；
- CoT-VLA。

因此真实场景下的相对优势还需要更广泛验证。

### 8.4 长时序规划能力仍有限

虽然 AffordanceVLA 在连续任务中表现出一定 emergent long-horizon execution，但其核心仍是单帧观测和短 action chunk 重规划。

论文也承认未来需要显式时间建模和长期记忆。

### 8.5 3D 表示的有效性仍有进一步验证空间

How2Act 在 LIBERO 和 CALVIN 中带来的提升相对有限，论文解释为这些任务多是简单桌面和两指夹爪任务。

虽然真实复杂任务中 How2Act 更有价值，但仍需要更多：

- 复杂 6-DoF 操作；
- 多视角；
- 遮挡；
- 非刚体；
- 双臂操作；

来验证 3D affordance 的普适性。

### 8.6 标注质量可能存在隐性偏差

虽然论文进行了严格抽检，但自动标注仍可能受到：

- VLM 指令理解偏差；
- grounding 模型偏差；
- SAM 分割边界误差；
- SAM-3D 重建误差；
- 合成数据分布偏差；

等因素影响。

这些偏差可能在更开放、更杂乱的真实环境中被放大。

# 总体评价

AffordanceVLA 是一篇较完整、工程实现充分且问题意识明确的 VLA 机器人操作论文。其核心贡献在于将 **结构化可供性预测** 作为视觉语言理解与动作控制之间的中间桥梁，并通过 Which2Act、Where2Act、How2Act 将机器人操作所需的“操作对象—交互位置—三维几何”显式建模。

从实验结果看，该方法在 LIBERO、CALVIN 和真实机器人任务上均取得了较强性能，尤其在语言敏感性、空间定位和复杂操作任务中表现突出。论文的主要局限在于训练和标注 pipeline 复杂、算力成本较高、真实机器人对比范围有限，以及长期规划能力仍未显式解决。

总体而言，该工作的重要启示是：

> 对于 VLA 机器人模型，单纯扩大数据或直接动作回归可能不足以解决视觉语言语义空间与物理控制空间之间的结构鸿沟；设计任务相关、空间锚定且动作耦合的中间表示，是提升泛化性、鲁棒性和可解释性的关键方向。

（完）
