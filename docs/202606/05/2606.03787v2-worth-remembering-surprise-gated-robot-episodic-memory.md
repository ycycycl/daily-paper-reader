## 摘要
执行通用任务的机器人需要能够将指令扎根于其过往经验之中，因为人类在下达任务时可能会提及值得注意的过去事件（例如，“带我去昨天发生化学品泄漏的地方”）。由于记忆容量限制使得存储所有过去事件不可行，机器人的长期记忆必须具有选择性，理想情况下只保留那些对未来任务具有高效用的情景。然而，对于通用机器人而言，未来任务通常并非事先给定。为选择具有通用实用性的记忆，我们提出将贝叶斯惊讶作为记忆形成的门控机制。我们提出了一种方法，在由 V-JEPA-2 提供的语义丰富且与部署无关的潜在空间中计算惊讶。通过使用我们的门控情景记忆来增强基于 4D 场景图的空间记忆，我们表明，在机器人问答任务中相较于最先进基准取得了一致提升；在时间、空间和二元问题上，相比以往机器人记忆方法性能提高了至少 12%，并且在事件分割任务中，以一种无监督的因果方法超过了有监督和非因果方法的性能。

## Abstract
Robots solving generalist tasks need to be able to ground instructions in their past experience, since humans may refer to notable past events when giving a task (e.g., ``Take me to where the chemical spill happened yesterday''). Since memory limits make storing all past events infeasible, long-term robot memory must be selective, ideally retaining only those episodes with high utility for future tasks. However, future tasks are not typically given a priori for generalist robots. To select generically useful memories, we propose Bayesian surprise as a gating mechanism for memory formation. We present an approach to compute surprise in a semantically rich deployment-agnostic latent space provided by V-JEPA-2. Using our gated episodic memory to augment 4D scene graph-based spatial memory, we show a consistent improvement over state-of-the-art benchmarks in robot question answering, outperforming prior robot memory methods by $\geq12\%$ for temporal, spatial, and binary questions, and surpassing the performance of supervised and non-causal methods with an unsupervised causal method in event segmentation tasks.

  - **什么值得存储？**
  - **什么时候形成情景记忆？**
  - **如何在不知道未来任务的情况下，选择对未来可能有用的经历？**

  - 固定时间间隔存储；
  - 随机采样存储；
  - 针对特定任务的人为触发规则；
  - 近乎无选择地保存所有观测。

## 2. 方法论：基于 V-JEPA-2 潜空间的惊讶门控情景记忆

### 2.1 核心思想

- 论文提出一种**因果、无监督、在线**的情景记忆形成机制。
- 系统从机器人连续视觉流中提取视频潜在表示，并计算当前观测相对于过去局部统计模型的“惊讶程度”。
- 当惊讶信号出现显著峰值时，系统将该时间点附近的图像片段保存为情景记忆。
- 这些情景记忆随后被加入 DAAAM 的 4D 场景图记忆系统中，供大语言模型 / 多模态智能体检索和推理。

---

### 2.2 为什么使用 V-JEPA-2 潜空间

论文没有直接在像素空间计算预测误差，原因是：

- 像素空间中存在大量随机变化，例如树叶晃动、光照闪烁、相机噪声等；
- 这些变化可能导致高预测误差，但并不一定是语义上重要或值得记住的事件；
- 机器人部署环境复杂多变，不能依赖每个场景重新训练的世界模型。

因此，作者使用 **V-JEPA-2** 作为视频表征模型：

- V-JEPA-2 通过自监督方式在大规模真实视频上训练；
- 它学习预测视频中的时空特征块；
- 其潜空间更关注语义连续性、运动、场景结构等对预测有用的信息；
- 相比原始像素，它更适合作为“语义惊讶度”的计算空间。

---

### 2.3 惊讶度计算流程

系统输入为机器人连续相机帧流。

#### 步骤 1：滑动窗口编码

- 在每个时间步，取最近的 \(M\) 帧作为一个因果滑动窗口；
- 论文中使用 \(M = 64\)；
- 将该窗口输入 V-JEPA-2 编码器；
- 对输出 token 进行池化，得到一个 \(D\) 维特征向量：
  \[
  z_t \in \mathbb{R}^D
  \]
- 文中 \(D = 1024\)。

相邻窗口共享 \(M-1\) 帧，因此 \(z_{t-1}\) 与 \(z_t\) 的变化主要来自新到达的一帧。

---

#### 步骤 2：拟合局部预测分布

- 系统在过去 \(W\) 个潜在特征上拟合一个对角高斯分布；
- 论文中 \(W = 64\)；
- 对每个维度单独估计均值和标准差：
  \[
  p_t = \prod_d \mathcal{N}(\mu_W[t,d], \sigma_W^2[t,d])
  \]

这个高斯分布被视为当前时刻的局部预测模型，即“根据过去观测，当前应该大致是什么样”。

---

#### 步骤 3：计算逐帧惊讶信号

论文使用一个 \(\ell_1\) 形式的鲁棒近似分数：

\[
s(t) = \frac{1}{D} \sum_{d=1}^{D}
\frac{|z_t[d] - \mu_W[t,d]|}{\sigma_W[t,d]}
\]

含义是：

- 如果当前潜在特征 \(z_t\) 与过去窗口统计均值差异很大；
- 且这种差异不能由过去波动解释；
- 则该时刻具有较高惊讶度。

作者进一步说明，在对角高斯假设下，这一分数与连续预测分布之间的贝叶斯 KL 散度相关。理论上，贝叶斯惊讶度可写为先验预测分布和观测后后验预测分布之间的 KL 散度：

\[
D_{KL}(p_{t+1} \| p_t)
\]

在滑动高斯近似下，其主导项与点态 surprisal 相关：

\[
D_{KL}(p_{t+1} \| p_t) \approx
\frac{1}{2W^2}
\sum_d
\frac{(z_t[d]-\mu_W[t,d])^2}{\sigma_W^2[t,d]}
\]

论文实际采用 \(\ell_1\) 形式，是为了增强对高维特征长尾分布的鲁棒性。

---

### 2.4 事件触发与阈值机制

系统寻找惊讶信号 \(s(t)\) 的局部峰值。若峰值超过鲁棒阈值，则触发情景记忆存储。

阈值定义为：

\[
\tau = \text{med}(s) + \gamma \cdot \frac{1}{\Phi^{-1}(0.75)} \cdot \text{MAD}(s)
\]

其中：

- \(\text{med}(s)\)：惊讶信号的中位数；
- \(\text{MAD}(s)\)：median absolute deviation，中位数绝对偏差；
- \(\frac{1}{\Phi^{-1}(0.75)} \approx 1.4826\)：高斯一致性校正因子；
- \(\gamma = 1.0\)：灵敏度参数。

随后使用 **非极大值抑制（NMS）** 去除密集重复峰值，避免同一个事件被多次存储。

---

### 2.5 情景记忆内容与检索

当某个时间点被判定为“值得记住”时，系统保存：

- 触发时间戳；
- 机器人位姿；
- 触发点附近的短视频片段；
- 惊讶分数；
- 用于后续检索的视觉-文本嵌入。

具体实现中：

- 每个情景记忆保存 **8 帧**；
- 每帧使用 **Perception-Encoder PE-Core-L14** 编码；
- 这些情景记忆作为额外层加入 **DAAAM** 的 4D 场景图记忆系统；
- 后续 LLM-agent 可以根据查询文本、位置、时间等信息检索相关 episode；
- 检索出的视觉片段以图像形式提供给多模态智能体辅助回答问题。

- **惊讶门控情景记忆可以提升机器人长期问答能力。**  
  在 OC-NaVQA 上，相比 DAAAM：
  - 问题准确率从 0.711 提升到 0.796；
  - 位置误差从 41.75 m 降至 36.57 m；
  - 时间误差从 1.792 min 降至 1.510 min。

- **相比固定间隔或随机存储，surprise gating 更有效。**  
  在相同记忆预算下：
  - uniform EM 和 random EM 虽然提升了部分问题准确率；
  - 但可能引入冗余视觉信息，干扰时间和空间推理；
  - surprise EM 在准确率、空间误差和时间误差上整体最好。

- **惊讶事件往往对应人类更可能引用的事件。**  
  例如开门、车辆停止、人物动作、场景突变等事件，更容易出现在人类问题和指令中。

- **在 Kinetics-GEBD 上，本文方法作为在线无监督方法取得很强结果。**  
  平均 F1 达到 0.833，超过多种监督和离线方法。

- **但在 TAPOS 上表现不如部分监督方法。**  
  说明通用视觉惊讶度更适合开放场景中的自然事件边界，而对强领域语义边界不一定足够。

---

## 7. 优点与亮点

- **提出了清晰且有认知科学依据的记忆选择标准。**  
  论文将贝叶斯惊讶度、预测误差、事件分割理论和机器人情景记忆结合起来，动机自然且理论基础较强。

- **方法是无监督、因果、在线的。**  
  不需要未来帧，不需要事件标签，也不需要针对每个部署环境重新训练。

- **使用 V-JEPA-2 潜空间避免像素级噪声。**  
  该设计使惊讶度更偏语义和时空结构，而不是受低层随机变化支配。

- **与现有机器人记忆系统 DAAAM 兼容。**  
  方法不是替代空间记忆，而是为 4D 场景图增加情景事件层，架构上具有模块化优势。

- **实验中有合理消融。**  
  与 uniform/random episodic memory 在相同记忆预算下比较，较好地说明“选择什么存”比“多存一点”更关键。

- **跨任务验证。**  
  同一个惊讶信号既提升机器人问答，也能用于通用视频事件边界检测，说明方法具有一定通用性。

- **存储效率较高。**  
  文中报告在 \(\gamma=1.0\) 下平均每分钟保存 1.28 个 episode，大约只保存 1.7% 的帧，显著减少长期记忆负担。

## 8. 不足与局限

- **预测模型仍较简单。**  
  虽然使用 V-JEPA-2 提供潜在表示，但在线预测分布只是滑动窗口上的对角高斯模型，表达能力有限，无法建模复杂长期动态和多模态分布。

- **V-JEPA-2 本身不是严格因果预测模型。**  
  V-JEPA-2 训练时采用非因果自监督视频预测，论文在线阶段用高斯模型作为因果代理。这可能限制惊讶度的理论一致性和实际精度。

- **重复出现的显著事件可能反复触发。**  
  因为方法基于滑动窗口局部统计，如果一个对象或事件周期性出现，只要它再次相对局部上下文显著，就可能重复写入。作者也指出未来可加入 habituation，即习惯化机制。

- **检索机制仍较基础。**  
  当前主要使用图文嵌入检索 8 帧视觉片段。对于需要理解事件动态过程的问题，单帧或短片段图文相似度可能不足，未来可能需要文本-视频嵌入或更强的时序检索。

- **缺少算力和实时性评估。**  
  论文没有报告硬件资源、吞吐量、延迟、显存和长期部署成本，因此实际机器人系统能否实时运行仍需进一步验证。

- **机器人问答数据规模有限。**  
  OC-NaVQA 只有 7 条长序列，虽然具有挑战性，但仍不足以完全证明不同机器人平台、场景和任务上的泛化能力。

- **依赖强大的多模态 LLM。**  
  下游问答性能与 GPT-5-mini 的推理能力、提示设计和检索接口密切相关。因此实验结果并不完全反映记忆门控机制本身的贡献。

- **token 成本增加。**  
  论文提到加入情景记忆后平均 token 使用增加约 13%，并且需要多模态 LLM 对视觉 episode 进行推理。

- **在强领域语义边界任务上表现有限。**  
  TAPOS 实验显示，通用惊讶度不一定能捕捉需要专业知识定义的事件边界，例如体育动作阶段划分。
