## 摘要
执行长时程视觉操作任务的机器人观测到的是高维图像，但成功的规划依赖于与动作相关的事实：当前可以执行什么，以及执行后会发生什么变化。一个有用的规划表示应当摒弃无关的视觉细节，同时保留动作的适用性和效果。经典任务规划器通过带有前提条件和效果的符号算子来利用这种结构，但从原始视觉经验中获得此类表示仍然具有挑战性。我们研究一种视觉任务规划设置，其中机器人仅接收图像转移：当前图像、已执行的高层动作以及结果图像。在测试时，给定起始图像和目标图像，机器人必须生成一系列能够到达目标的高层动作。为解决这一问题，我们提出 STRIPS-WM，一个用于直接从视觉转移中学习图像扎根的 STRIPS 风格世界模型的框架。STRIPS-WM 首先从图像中归纳出一个有限的抽象转移图，然后学习潜在二值谓词以及每个动作标签对应的一个扎根命题算子。学习到的算子构成一个具有稀疏前提条件和添加/删除效果的符号动作模型。最后，将学习到的谓词蒸馏到视觉编码器中，从而能够直接基于新的起始图像和目标图像进行经典规划。视觉重排任务上的实验表明，与所测试的视觉 rollout、潜在图搜索和潜在符号基线方法相比，STRIPS-WM 提高了从图像到规划的成功率。

## Abstract
Robots performing long-horizon visual manipulation observe high-dimensional images, but successful plans depend on action-relevant facts: what can be done now and what changes afterward. A useful planning representation should discard irrelevant visual details while preserving action applicability and effects. Classical task planners exploit this structure through symbolic operators with preconditions and effects, but obtaining such representations from raw visual experience remains challenging. We study a visual task-planning setting in which a robot receives only image transitions: the current image, executed high-level action, and the resulting image. At test time, given a start image and a goal image, the robot must produce a sequence of high-level actions that reaches the goal. To address this problem, we introduce STRIPS-WM, a framework for learning image-grounded STRIPS-style world models directly from visual transitions. STRIPS-WM first induces a finite abstract transition graph from images, then learns latent binary predicates and one grounded propositional operator per action label. The learned operators form a symbolic action model with sparse preconditions and add/delete effects. Finally, the learned predicates are distilled into a visual encoder, enabling classical planning directly from novel start and goal images. Experiments on visual rearrangement tasks show that STRIPS-WM improves image-to-plan success over the tested visual rollout, latent graph-search and latent-symbolic baselines.

## 2. 方法论：核心思想、关键技术细节与流程

论文提出的方法称为 **STRIPS-WM**，完整流程分为三个阶段。

### 2.1 阶段一：从图像转移学习图像扎根的任务图

- 输入数据为视觉转移数据集：
  \[
  D_O = \{(o_i, a_i, o'_i)\}_{i=1}^{N}
  \]
  其中：
  - \(o_i\)：动作前图像；
  - \(a_i\)：不带语义的高层动作标识符；
  - \(o'_i\)：动作后图像。

- 目标是学习一个离散状态分配函数：
  \[
  \alpha: O \rightarrow \hat{S}
  \]
  将图像映射到有限抽象状态节点。

- 技术细节：
  - 使用 **student encoder** 编码当前图像；
  - 使用 **EMA teacher encoder** 编码后继图像；
  - 使用 **FSQ（Finite Scalar Quantization）** 将连续表示量化为离散代码；
  - 使用动作条件化动力学模型，根据当前离散代码和动作标识预测下一代码；
  - 使用逆动力学模型，从当前代码和后继代码预测执行动作，以防表示坍缩。

- 训练目标包含：
  - 动力学预测损失；
  - 逆模型动作分类损失；
  - 量化 commitment loss；
  - 数据增强一致性损失；
  - 用于减少潜在非确定性的 separation loss。

- 训练完成后，每个图像被映射为离散代码，不同代码对应任务图节点：
  \[
  \hat{s} = \alpha(o)
  \]

- 对每个视觉转移：
  \[
  (o_i, a_i, o'_i)
  \]
  得到抽象转移：
  \[
  (\alpha(o_i), a_i, \alpha(o'_i))
  \]

- 最终形成图像扎根任务图：
  \[
  \hat{G} = (\hat{S}, \hat{E})
  \]

---

### 2.2 阶段二：用 CP-SAT 学习谓词和 STRIPS 算子

任务图仍然只是“状态节点 + 动作边”的扁平图，没有谓词、前提条件和动作效果。论文进一步用 CP-SAT 求解器学习符号结构。

- 对每个图节点 \(\hat{s}\)，学习一个二值谓词向量：
  \[
  \hat{x}_{\hat{s}} \in \{0,1\}^m
  \]

- 对每个动作标识符 \(a\)，学习一个 grounded propositional STRIPS 算子：
  \[
  \hat{\omega}_a = (\text{pre}^+_a, \text{pre}^-_a, \text{add}_a, \text{del}_a)
  \]

- 其中：
  - \(\text{pre}^+_a\)：正前提；
  - \(\text{pre}^-_a\)：负前提；
  - \(\text{add}_a\)：添加效果；
  - \(\text{del}_a\)：删除效果。

#### 正证据约束：观测边必须可解释

对于每条观测边：
\[
(\hat{s}_0, a, \hat{s}_1) \in \hat{E}
\]

要求：

- 动作 \(a\) 在源状态 \(\hat{x}_{\hat{s}_0}\) 中可执行；
- 执行动作后的状态等于目标状态：
  \[
  \tau_{\hat{\omega}_a}(\hat{x}_{\hat{s}_0}) = \hat{x}_{\hat{s}_1}
  \]

其中 STRIPS 转移形式为：
\[
x' = \text{add}_a \vee (x \wedge \neg \text{del}_a)
\]

即：
- add 中的谓词变真；
- del 中的谓词变假；
- 其他谓词保持惯性不变。

#### 负证据约束：未观测动作应尽量不可执行

仅靠观测边会导致一个退化解：动作没有前提条件，因此处处可执行。为避免这种问题，论文引入 trusted missing state-action pairs：

\[
\hat{N} = \{(\hat{s}, a) \mid \hat{s} \in \hat{S}_{trust}, \nexists \hat{s}' \text{ s.t. } (\hat{s}, a, \hat{s}') \in \hat{E}\}
\]

含义是：在可信状态中，如果某动作没有被观测到，则将其视作负证据，即该动作在该状态中不应可执行。

对于每个负证据对 \((\hat{s}, a)\)，要求至少有一个前提条件被违反，除非使用 slack 变量。

#### Slack 与鲁棒性

- 论文引入两类 slack：
  - transition slack \(\xi\)：允许某些观测边的效果解释存在局部错误；
  - applicability slack \(\eta\)：允许某些负证据不被完全满足。

- 这使模型可以处理视觉抽象中的噪声、状态混叠或过度划分。

#### 词典序优化目标

论文使用词典序目标依次最小化：

1. transition slack；
2. applicability slack；
3. 动作效果数量；
4. 动作前提数量。

其直觉是：
- 优先准确解释观测转移；
- 其次尽量排除未观测动作；
- 再偏好稀疏效果；
- 最后偏好稀疏前提。

### 2.3 阶段三：训练视觉谓词分类器并进行测试时规划

CP-SAT 学到的是图节点上的谓词赋值，不能直接用于新图像。因此论文训练视觉谓词分类器：

\[
b_\phi: O \rightarrow [0,1]^m
\]

- 输入：图像；
- 输出：每个谓词为真的概率。

训练标签来自第二阶段学到的谓词向量：
\[
\hat{x}_{\hat{s}}
\]

训练损失为多标签二元交叉熵，并加入 logit margin 正则，使预测远离 0.5 的二值化阈值。

测试时：

1. 输入起点图像 \(o_{start}\) 和目标图像 \(o_{goal}\)；
2. 用视觉谓词分类器得到：
   \[
   \hat{x}_{start}, \hat{x}_{goal}
   \]
3. 将概率按 0.5 阈值二值化；
4. 使用学习到的 STRIPS 算子集合 \(\hat{\Omega}\) 进行经典规划；
5. 输出动作序列：
   \[
   \pi = (a_1, \dots, a_H)
   \]

- 对真实世界任务的测试仍是离线图像和 ground-truth transition graph 执行评估，而非真实机器人闭环执行；
- 数据集规模较小到中等，未展示大规模开放场景泛化；
- 未充分评估对遮挡、光照变化、动作失败、随机动态的鲁棒性；
- 没有报告不同随机种子下的方差或置信区间；
- 部分评估依赖人工构建的 ground-truth transition graph；
- LatPlan 等 baseline 的调参公平性虽然有说明，但具体调参预算和实现细节仍不完全透明。

---

## 6. 主要结论与发现

论文的主要发现包括：

- STRIPS-WM 能够从纯视觉转移数据中学习图像扎根的 STRIPS 风格世界模型。
- 在 BlocksWorld 中，方法精确恢复了真实抽象：
  - ground-truth 状态数为 16；
  - 学到的视觉状态数为 16；
  - 学到的谓词状态数为 16；
  - transition slack 和 applicability slack 均为 0。
- 在 DinnerTable 中，方法也恢复了 101 个真实任务状态，但存在少量 applicability slack，说明动作适用性模型仍有少量误差。
- 在 DinnerTable Real 中，视觉抽象阶段过度划分状态：
  - 学到 111 个视觉节点；
  - 真实任务状态为 71；
  - CP-SAT 最终将其折叠为 71 个谓词状态。
- STRIPS-WM 在长程规划成功率上显著优于 baseline。
- 在 BlocksWorld 和 DinnerTable Real 上，STRIPS-WM 达到 100% 规划成功率。
- 在 DinnerTable 模拟环境中，长 horizon 下出现失败，主要来自学到的 applicability model 中的真实错误。
- 负证据对学习有用前提条件至关重要：
  - 去掉负证据后，模型会出现严重的动作适用性假阳性；
  - 即使强制谓词状态数不少于真实状态数，也不能替代负证据。

---

## 7. 优点与亮点

### 7.1 方法层面

- **将视觉世界模型与经典 STRIPS 规划结合**：  
  不是单纯学习 latent dynamics，而是学习可用于经典规划器的谓词状态和动作算子。

- **无需人工谓词、对象标签或分割信息**：  
  训练只需要图像转移和动作 ID，不需要对象中心监督。

- **分阶段设计清晰**：
  1. 图像到抽象图；
  2. 抽象图到 STRIPS 模型；
  3. 图像到谓词分类器；
  4. 测试时经典规划。

- **CP-SAT 形式化较严谨**：  
  将谓词赋值、前提条件、add/delete 效果统一编码为二值约束，并提供了理论保证。

- **引入负证据学习动作不可用性**：  
  这是从观测转移学习前提条件的关键设计，实验也证明非常重要。

- **支持状态折叠与抽象修复**：  
  在 DinnerTable Real 中，视觉图过度划分，但 CP-SAT 谓词层可以将动作等价状态合并为正确任务状态。

- **相比扁平图搜索更紧凑**：  
  直接图搜索需要在节点和边上搜索，而 STRIPS-WM 使用谓词和动作算子，更适合长程组合规划。

---

### 7.2 实验层面

- 同时包含模拟和真实图像任务；
- 使用多个 baseline，覆盖 rollout、latent BFS、roadmap、latent symbolic planning；
- 评估时在 ground-truth transition graph 中执行计划，标准较严格；
- 提供负证据消融，能清楚解释方法成功的关键原因；
- 附录中给出较完整的 CP-SAT 编码和形式化证明。

## 8. 不足与局限

### 8.1 模型假设限制

- 假设环境是确定性的；
- 假设高层动作具有固定的前提条件和 add/delete 效果；
- 不处理随机动作结果、部分可观测状态或隐藏变量；
- 不建模动作失败概率。

### 8.2 表达能力限制

- 学到的是 **grounded propositional operators**，不是 lifted relational schemas；
- 因此不能自然泛化到：
  - 新数量的物体；
  - 新对象组合；
  - 新动作参数；
  - 更大规模的组合任务。
- 每个动作 ID 对应一个 grounded operator，动作库扩大时可能需要更多数据和更复杂的求解。

### 8.3 对数据覆盖的依赖

- 方法依赖学习到的任务图 \(\hat{G}\) 的质量；
- 如果视觉抽象出现状态混叠，可能导致错误谓词和错误动作效果；
- 如果某些状态的 outgoing action 覆盖不完整，负证据可能误导前提条件学习；
- trusted missing pairs 的可靠性是关键假设。

### 8.4 计算和扩展性问题

- CP-SAT 是组合优化，随着：
  - 状态数；
  - 动作数；
  - 谓词维度；
  - 负证据数量；
  - distinctness 约束数量  
  增长，求解复杂度可能快速上升。

- 论文只在最大约百级状态规模上展示结果，尚未证明能扩展到更大任务图。

### 8.5 实验覆盖不足

- 实验集中在视觉重排任务；
- 未覆盖导航、工具使用、多步骤装配、人机交互等更开放任务；
- 真实实验是基于真实图像数据和图评估，不是完整真实机器人在线执行；
- 未报告硬件资源和训练耗时，不利于复现成本评估；
- 未报告多随机种子统计，结果稳定性还需进一步验证。

### 8.6 感知泛化风险

- 视觉谓词分类器直接学习 CP-SAT 产生的二值谓词标签；
- 如果训练图像分布有限，面对新视角、新光照、新背景、新遮挡时可能失效；
- 谓词本身是 latent 的，虽然可用于规划，但不一定具有人类可解释语义。

---

## 总结

STRIPS-WM 是一篇将视觉表示学习、符号动作模型学习和经典规划结合起来的工作。它从图像动作转移中先学习离散任务图，再通过 CP-SAT 学习二值谓词与 STRIPS 算子，最后训练视觉谓词分类器，使机器人能够从新起点和目标图像直接进行经典规划。实验表明，该方法在视觉重排任务中优于视觉 rollout、latent graph search 和 LatPlan 类方法，尤其在长程规划和真实图像状态过度划分时表现突出。其主要贡献在于提供了一条从原始视觉经验到可规划符号世界模型的完整路径。不过，该方法仍受限于确定性、命题式、grounded 动作模型以及对数据覆盖和 CP-SAT 扩展性的依赖，未来需要进一步验证其在更大规模、更开放、更随机的真实机器人任务中的适用性。
