## 摘要
在充满威胁的环境中为自主车辆进行路径规划是一项基础性挑战，因为即使在最简单的场景中，该问题也是非线性且非凸的。尽管传统最优控制方法可用于寻找理想路径，但其计算时间通常过长，难以满足实时决策需求。为解决这一挑战，我们提出一种基于深度确定性策略梯度（Deep Deterministic Policy Gradient, DDPG）的方法，并将威胁建模为可能存在的多个圆形“禁行”区域。若车辆在任何时刻进入该受限区域，或未能到达目的地邻域，则任务被视为失败。DDPG 智能体在仿真环境中通过试错进行训练，学习从其当前状态（位置和航向）到一系列可行动作的直接映射，从而引导智能体安全到达目的地。奖励函数由三部分组成：（a）以最终目的地为中心的吸引场；（b）若干以圆形障碍物圆心为中心的排斥场；以及（c）对控制能量消耗（航向变化幅度）的惩罚，该惩罚间接偏好直线路径。DDPG 利用这些激励来训练智能体，以寻找尽可能大的起始点集合，使得从这些起始点出发能够保证存在一条通往目的地的安全路径。这为任务规划提供了关键信息，可预先判断从给定起始点出发的任务是否可实现，从而辅助任务前规划活动。该方法在仿真中得到验证，并将 DDPG 方法与传统最优控制（伪谱）方法进行了比较。结果表明，基于学习的智能体能够生成有效路径，同时速度显著更快，因此更适合实时应用。

## Abstract
Path-planning for autonomous vehicles in threat-laden environments is a fundamental challenge because the problem is nonlinear and nonconvex even in simplest scenarios. While traditional optimal control methods can be used to find ideal paths, the computational time is often too slow for real-time decision-making. To solve this challenge, we propose a method based on Deep Deterministic Policy Gradient (DDPG) and model the threat as possibly multiple circular 'no-go' zones. A mission is regarded as a failure if the vehicle enters this restricted zone at any time or does not reach a neighborhood of the destination. The DDPG agent is trained through trial and error in a simulated environment, learning a direct mapping from its current state (position and heading) to a series of feasible actions that guide the agent to safely reach its destination. The reword function has three parts: (a) an attractive field centered at the final destination, (b) some repulsive fields centered at the origins of circular obstacles, and (c) a penalty of control energy consumption (the magnitude of heading change) that indirectly in favor for straight path. The DDPG trains the agent using these incentives to find the largest possible set of starting points wherein a safe path to the destination is guaranteed. This provides critical information for mission planning, showing beforehand whether a task is achievable from a given starting point, assisting pre-mission planning activities. The approach is validated in simulation. A comparison between the DDPG method and a traditional optimal control (pseudo-spectral) method is carried out. The results show that the learning-based agent produces effective paths while being significantly faster, making it a better fit for real-time applications.

## 2. 方法论：核心思想、关键技术细节与算法流程

### 2.1 总体思路

- 论文采用 **DDPG** 作为核心强化学习框架。
- 智能体在仿真环境中通过试错学习策略：
  - 输入：当前位置、当前航向、时间步等状态信息。
  - 输出：连续动作，即航向变化或新的航向控制量。
- 环境中设置圆形障碍物或禁区。
- 如果智能体进入禁区，或未能到达目标附近，则任务失败。
- 奖励函数结合：
  1. 目标点吸引势场；
  2. 障碍物排斥势场；
  3. 航向变化能耗惩罚；
  4. 每步时间惩罚。

### 2.2 DDPG 结构

- 使用典型的 Actor-Critic 结构：
  - **Actor 网络**：根据状态输出确定性动作。
  - **Critic 网络**：评估状态-动作对的长期回报，即 Q 值。
- Actor 策略形式为：

  \[
  A_\theta = \mu(S;\theta_A)
  \]

- 为了探索，动作中加入噪声：

  \[
  A_\theta = \mu(S;\theta_A) + N
  \]

- 噪声采用高斯噪声或 Ornstein-Uhlenbeck 噪声的思想。
- 论文中提到训练早期使用较大的标准差进行探索，标准差大致选在 **3 到 5 度**范围内；训练后期可降低噪声以进行利用和微调。

### 2.3 状态、动作与动力学更新

- 状态包含：
  - 智能体二维位置 \(x, y\)；
  - 当前或上一时刻航向；
  - 时间步信息。
- 动作主要对应航向变化。
- 位置更新形式为：

  \[
  P_{t+1} = P_t + |V|[\cos(\phi_{t+1}), \sin(\phi_{t+1})]^T T_s
  \]

- 其中：
  - \(T_s = 0.25\) 秒；
  - \(V\) 为速度；
  - \(\phi_{t+1}\) 为更新后的航向。

### 2.4 奖励函数设计

论文奖励函数是方法的关键。主要由以下几部分组成：

#### 目标吸引势场

- 以目标点为中心构造高斯型人工势场。
- 如果智能体从 \(P_t\) 移动到 \(P_{t+1}\) 后更接近目标，则获得正向奖励。
- 使用归一化高斯分布差值表示：

  \[
  \tilde{N}(P_{t+1}, P_f, \sigma_1) - \tilde{N}(P_t, P_f, \sigma_1)
  \]

#### 障碍物排斥势场

- 对圆形禁区中心构造高斯型排斥势场。
- 如果智能体靠近障碍物，则受到惩罚。
- 单障碍物情况下，排斥项形式类似：

  \[
  -w_2[\tilde{N}(P_{t+1}, P_c, \sigma_2)-\tilde{N}(P_t, P_c, \sigma_2)]
  \]

- 多障碍物场景中，对多个障碍物中心的排斥势场进行求和。

#### 控制能耗惩罚

- 使用航向变化幅度平方作为能耗惩罚：

  \[
  -w_3 |A_t|^2
  \]

- 该项鼓励较平滑、较直的路径，间接缩短路径长度和降低控制消耗。

#### 每步时间惩罚

- 每一步加入固定惩罚，例如 \(-10\)，以鼓励智能体尽快到达目标，减少路径耗时。

### 2.5 智能初始航向设计

论文的一个重要改进是引入 **smart initial heading**。

- 如果起点到目标点的直线不穿过禁区，则初始航向直接指向目标：

  \[
  \phi_0 = \tan^{-1}\frac{y_f-y_0}{x_f-x_0}
  \]

- 如果起点到目标点的直线穿过圆形禁区，则计算从起点到圆形障碍物的两个切点。
- 然后选择与目标点距离较短的切点作为中间参考方向。
- 初始航向指向该切点，从而避免智能体在早期直接朝障碍物运动。
- 作者给出了圆心在原点、圆心不在原点等不同情况下的切点计算公式，并将一般形式用于多障碍物情形。

### 2.6 算法流程概括

- 初始化环境、目标点、障碍物位置与半径。
- 初始化 DDPG 的 Actor 和 Critic 网络。
- 每个训练 episode：
  - 调用 reset 函数，设置起点和智能初始航向；
  - 智能体根据当前状态输出动作；
  - 在动作中加入探索噪声；
  - 根据动力学更新位置和航向；
  - 根据奖励函数计算即时奖励；
  - 判断是否到达目标或失败；
  - 更新 Actor 和 Critic 网络参数。
- 训练完成后：
  - 在网格化起点或指定起点上测试；
  - 判断能否安全到达目标；
  - 绘制可行起点集合和路径示例。

  - 是一个简单但有效的启发式改进。
- **关注可行起点集合**：
  - 论文不仅展示单条路径，还尝试识别哪些起点能够安全到达目标；
  - 这对任务前规划有实际意义。
- **从单障碍物扩展到多障碍物**：
  - 验证了方法在更复杂场景中的一定适用性。
- **融合人工势场与强化学习**：
  - 人工势场提供了形状良好的奖励信号；
  - DDPG 学习连续控制策略；
  - 二者结合可以缓解稀疏奖励问题。

- **量化实验不足**：

- **训练细节披露不足**：
  - 未说明网络结构细节、超参数、训练 episode 数、学习率、batch size、replay buffer 大小等。

- **场景较理想化**：
  - 障碍物均为静态圆形禁区；
  - 未考虑动态障碍物；
  - 未考虑复杂地形、不确定环境、传感器噪声或真实车辆动力学约束。
