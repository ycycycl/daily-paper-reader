## 摘要
视频世界模型旨在模拟可控的视觉环境，但长时域展开依赖于观测离开模型原生上下文窗口后模型所保留的信息。显式记忆保存帧或在线三维重建结果，可能受到启发式检索错误、冗余外观存储或重建伪影的影响。隐式记忆将历史压缩为紧凑状态，但现有设计并未受到显式约束以编码跨视角场景几何。我们提出 GIM-World，一种面向视频世界模型的几何感知隐式记忆框架。一个轻量级 Transformer 编码器将可变长度历史压缩为固定大小的记忆 token；一个可由相机查询的几何头在训练期间将来自冻结基础模型的三维场景结构蒸馏到记忆中；一种信息引导的剪枝规则在历史增长时保持编码成本有界。几何教师在推理时被丢弃，仅保留轻量级记忆模块。在 MIND 上的实验表明，与显式记忆和隐式记忆基线相比，GIM-World 能更好地保持长时域的几何一致性和视觉一致性。

## Abstract
Video world models aim to simulate controllable visual environments, but long-horizon rollouts depend on what the model remembers after observations leave its native context window. Explicit memories retain frames or online 3D reconstructions, which can suffer from heuristic retrieval errors, redundant appearance storage, or reconstruction artifacts. Implicit memories compress history into a compact state, but existing designs are not explicitly constrained to encode cross-view scene geometry. We propose GIM-World, a geometry-aware implicit memory framework for video world models. A lightweight transformer encoder compresses variable-length history into fixed-size memory tokens, a camera-queryable geometry head distills 3D scene structure from a frozen foundation model into the memory during training, and an information-guided pruning rule keeps encoding cost bounded as history grows. The geometry teacher is discarded at inference, leaving a lightweight memory module. Experiments on MIND show that GIM-World better preserves long-horizon geometric and visual consistency than both explicit- and implicit-memory baselines.

    - 历史帧检索依赖启发式规则，如视觉相似度或相机重叠；

    - 保存大量外观信息存在冗余；
    - 在线 3D 重建会增加推理开销，并可能受到动态物体、遮挡、弱纹理等影响。

  **2. 隐式记忆（Implicit Memory）**
  - 将历史压缩成固定大小的紧凑状态。
  - 优点是轻量、长度不随历史增长。
  - 但现有隐式记忆通常只依赖未来帧生成损失训练，并没有显式约束其编码跨视角三维几何结构，因此容易记住短期外观而非稳定世界状态。

  > 几何结构不应只作为生成器的输入、监督信号或后处理奖励，而应该成为“记忆状态”本身需要编码的内容。

## 2. 方法论：核心思想、关键技术与流程

## 2.1 整体框架

GIM-World 包含三个核心组成部分：

1. **隐式记忆编码器（Implicit Memory Encoder）**
   - 将可变长度历史观测压缩为固定数量的 memory tokens。

2. **相机可查询的几何监督头（Camera-Queryable Geometry Head）**
   - 训练阶段使用冻结的 3D 基础模型 VGGT 作为教师，将三维几何知识蒸馏进 memory tokens。
   - 推理阶段丢弃该几何头和 VGGT，仅保留轻量记忆模块。

3. **信息引导的历史剪枝（Information-Guided Pruning）**
   - 在历史变长时，选择最有信息量的一部分历史帧输入记忆编码器，控制编码成本。

---

## 2.2 问题形式化

在自回归世界模型中，给定历史观测：

\[
H_t = \{(x_i, c_i)\}_{i=1}^{t}
\]

其中：

- \(x_i\)：历史帧或其 VAE latent；
- \(c_i\)：对应相机位姿。

模型根据历史和未来动作生成未来 \(K\) 帧：

\[
x_{t+1:t+K} \sim p_\theta(\cdot | H_t, a_{t+1:t+K})
\]

但直接使用完整历史 \(H_t\) 会随着时间增长而不可扩展。因此，记忆模型使用一个 memory operator：

\[
m_t = M(H_t)
\]

将历史压缩成固定大小记忆状态，再生成未来：

\[
p_\theta(x_{t+1:t+K} | m_t, a_{t+1:t+K})
\]

GIM-World 的目标是设计一个既轻量又能编码跨视角几何结构的 \(M\)。

---

## 2.3 隐式记忆编码器

### 输入表示

- 历史帧先经过视频 VAE 和 diffusion backbone 的 patch embedding，得到 patch tokens：

\[
H = \{h_{i,p}\}_{i=1,p=1}^{T,P}
\]

其中 \(T\) 是历史帧数，\(P = H_p W_p\) 是每帧 patch 数。

- 相机信息只注入历史 tokens：

\[
\tilde{h}_{i,p} = h_{i,p} + E_c(c_i)
\]

其中 \(E_c\) 是线性相机嵌入。

- 记忆 queries 是固定数量的 learnable tokens：

\[
Q_0 \in \mathbb{R}^{N_m \times d}
\]

- 编码器输入为：

\[
Z_0 = [Q_0; \tilde{H}]
\]

### 记忆编码方式

- 编码器采用轻量 Transformer，仅使用两层 block。
- 每层包含：
  - compact self-attention；
  - full-resolution feed-forward network。

更新过程为：

\[
Z \leftarrow Z + Expand(Attn(Compact(Z)))
\]

\[
Z \leftarrow Z + FFN(Z)
\]

其中：

- `Compact` 将每帧 patch token 按 \(s \times s\) 分块压缩；
- `Expand` 将 attention 输出恢复到原分辨率；
- attention 在压缩 token 上运行以降低成本；
- FFN 保持全分辨率，避免损失细节表达能力。

最终取输出序列前 \(N_m\) 个 token 作为 memory tokens：

\[
m = Z[:N_m]
\]

这些 memory tokens 与目标 latents 沿时间维拼接，作为 DiT 生成器的条件。

## 2.4 相机可查询的几何监督

这是论文最关键的设计之一。

### 背景问题

已有方法如 Geometry Forcing 通常将 VGGT 等 3D 基础模型的特征直接对齐到生成器内部特征上。但 GIM-World 的 memory tokens 是压缩后的无序隐式状态，并不与输入帧或空间 patch 一一对应。

如果强行做 token-to-token 对齐，会破坏隐式记忆的优势，使其退化成帧索引缓存。

### 作者的做法

GIM-World 让 memory tokens 学会回答：

> “从某个历史相机视角看，场景的三维几何特征应该是什么？”

训练时随机采样一个历史帧 \(i\)，取其相机 \(c_i\)，构造每个 patch 的 ray query。

对于 patch \((u,v)\)，其世界空间射线表示为：

\[
\rho(c_i)_{u,v} = [o_i, d_{i,u,v}] \in \mathbb{R}^6
\]

其中：

- \(o_i\)：相机原点；
- \(d_{i,u,v}\)：该 patch 对应的归一化射线方向。

然后经过 MLP 和二维位置编码得到查询：

\[
q_{i,u,v} = MLP_{ray}(\rho(c_i)_{u,v}) + e_{u,v}
\]

几何头通过：

1. cross-attention：用 ray queries 查询 memory tokens；
2. self-attention：在 patch 网格内交换上下文；
3. FFN：投影到 VGGT 特征维度；

得到预测几何特征：

\[
\hat{G}_i = FFN(SelfAttn(CrossAttn(q_i, m)))
\]

然后与冻结 VGGT 提取的真实几何特征 \(G_i\) 做 per-patch cosine loss：

\[
L_{geo} = 1 - \frac{1}{H_g W_g}\sum_{u,v} cos(\hat{G}_{i,u,v}, G_{i,u,v})
\]

### 关键意义

- 记忆不是简单保存外观，而是被训练成一个可以由相机射线查询的几何世界状态。
- 几何教师 VGGT 只在训练阶段使用，推理阶段完全丢弃。
- 因此推理时不会引入额外 3D 模型开销。

## 2.5 信息引导的历史剪枝

虽然 memory tokens 数量固定，但编码器处理历史 tokens 的成本仍会随历史长度增长。

因此作者在编码前先剪枝历史帧，只保留最多 \(K\) 个最有信息量的历史观测。

目标是选择子集 \(S\)，最大化它与未选择历史之间的互信息：

\[
S^\star = \arg\max_S I(S; H \setminus S), \quad |S| \le K
\]

作者使用 Gaussian Process 建模历史帧之间的相关性，其 kernel 同时考虑：

- 相机位置距离；
- 相机朝向夹角；
- 时间距离。

kernel 为：

\[
k(c_i, c_j) = exp\left(
-\frac{\|p_i-p_j\|_2^2}{2\sigma_p^2}
-\frac{\angle(f_i,f_j)^2}{2\sigma_r^2}
-\frac{(t_i-t_j)^2}{2\sigma_t^2}
\right)
\]

其中：

- \(p_i\)：相机位置；
- \(f_i\)：相机前向方向；
- \(t_i\)：时间戳。

该互信息目标是 submodular，因此可以使用贪心算法近似优化。

贪心选择标准是 posterior variance 的 log-ratio：

\[
s_t = \arg\max_{h \in \bar{S}_{t-1}}
\frac{1}{2}
\log
\frac{\sigma^2(h|S_{t-1})}
{\sigma^2(h|\bar{S}_{t-1}\setminus \{h\})}
\]

直观解释：

- 如果当前已选集合无法预测某一帧，说明它提供新信息；
- 如果其他未选帧可以很好预测该帧，说明它具有代表性；
- 因此该策略倾向选择既新颖又能代表未选历史的帧。

## 2.6 训练目标与推理流程

训练使用 flow matching 损失：

\[
L_{FM} = \mathbb{E}[\|v_\theta(z_t,t,m,a)-v^\star\|_2^2]
\]

整体目标为：

\[
L = L_{FM} + \lambda L_{geo}
\]

其中论文设置：

\[
\lambda = 0.05
\]

训练时联合优化：

- 记忆编码器；
- 几何监督头；
- diffusion backbone。

推理时：

- 输入历史经过剪枝和记忆编码器；
- 得到固定大小 memory tokens；
- memory tokens 与动作条件一起输入 DiT；
- 几何头和 VGGT 均被丢弃。

## 6. 主要结论与发现

论文的主要结论包括：

- **几何感知隐式记忆可以显著提升长时域视频世界模型的一致性。**
- 将三维几何监督施加到 memory state 本身，比只对 generator 特征施加几何约束更有效。
- GIM-World 在 MIND 第一人称和第三人称测试集中均取得最佳综合结果。
- 在三维几何一致性上提升尤其明显：
  - 第一人称 reprojection score 达到 81.70；
  - 第三人称 reprojection score 达到 87.10；
  - 明显优于最强可比 baseline。
- 与 SSM 等隐式记忆相比，GIM-World 不仅保持固定大小记忆的优势，还能更稳定地编码跨视角场景结构。
- 与显式记忆相比，GIM-World 避免了启发式检索、冗余外观存储和在线重建误差。
- 信息引导剪枝比均匀采样和相机空间 FPS 更有效。
- Compact attention 设计在大幅降低记忆编码成本的同时，基本保持了 full attention 的质量。
- 几何教师和几何头在推理阶段可完全丢弃，使方法具备较好的推理效率。

## 7. 优点与亮点

## 7.1 方法设计亮点

- **将几何监督作用于记忆，而不是仅作用于生成器。**
  - 这是本文区别于 Geometry Forcing、VideoGPA、Epipolar-DPO 等方法的核心创新。

- **camera-queryable memory 设计合理。**
  - 通过相机射线查询 memory，使记忆学习成为一个隐式三维世界状态；
  - 避免了 memory tokens 与帧 patch 的硬对齐。

- **训练时使用 3D foundation model，推理时不使用。**
  - 兼顾了几何知识蒸馏和推理效率。

- **固定大小 memory tokens。**
  - 适合长时域 rollout，不会随着历史无限增长。

- **信息引导剪枝有明确理论依据。**
  - 使用 Gaussian Process 和互信息 sensor placement 目标；
  - 比简单时间采样或相机 FPS 更 principled。

- **compact attention 设计实用。**
  - attention 分支降采样，FFN 保持全分辨率；
  - 在效率和质量之间取得较好平衡。

---

## 7.2 实验设计亮点

- 同时比较显式记忆和隐式记忆方法；
- 评估指标覆盖面较广；
- 对重新训练 baseline 使用统一 backbone 和 memory budget；
- 消融实验直接对应论文三大核心模块；
- 定性结果展示了长达 1000 帧的重访一致性，直观体现记忆能力。

---

## 8. 不足与局限

## 8.1 实验覆盖局限

- 实验主要集中在 MIND benchmark。
  - MIND 虽然是视频世界模型长时记忆 benchmark，但仍属于特定数据分布。
  - 缺少在真实世界视频、自动驾驶、多机器人仿真、室内导航等数据上的验证。

- 第一人称和第三人称虽然覆盖两种视角，但仍不足以证明方法在所有交互场景中泛化。

---

## 8.2 对教师模型的依赖

- 训练阶段依赖冻结 VGGT 提供几何特征。
- 如果 VGGT 在某些场景中几何估计错误，可能将错误几何蒸馏进 memory。
- 对于动态物体、透明物体、弱纹理区域或复杂光照，3D teacher 的可靠性可能影响最终结果。
- 论文没有系统分析 VGGT teacher 质量对 GIM-World 的影响。

---

## 8.3 真实动态场景能力仍不明确

- MIND 数据来自 Unreal Engine 5，具有较强仿真特征。
- 对于真实世界中复杂动态物体、非刚体、人群、多智能体交互等情况，本文没有充分验证。
- 几何监督主要针对静态或相对稳定场景结构，对动态内容的长期记忆机制仍不清楚。

---

## 8.4 评价指标可能存在代理偏差

- 几何一致性依赖 Depth Anything 3 估计深度和相机。
- 动作准确性依赖 ViPE 恢复轨迹。
- 这些指标本身也是模型估计结果，可能存在误差或偏差。
- 论文没有提供人工评估或真实几何 ground truth 的大规模验证。

---

## 8.5 复现信息不完整

- 没有报告 GPU 型号、GPU 数量、训练时长等关键算力信息。
- 虽然给出了训练步数、学习率和 batch size，但完整复现成本仍不明确。
- 未报告随机种子和多次实验方差，结果稳定性尚需进一步验证。

---

## 8.6 长期扩展性仍需进一步检验

- 论文展示了 1000-frame qualitative rollout，但主定量评估仍基于 MIND 设置。
- 对更长时间、更多重访路径、更大空间地图的扩展性没有系统定量分析。
- 固定大小 memory 在极大规模场景中是否会出现容量瓶颈，仍是开放问题。

---

# 总体评价

GIM-World 是一篇围绕“视频世界模型长期记忆一致性”的方法型论文。其核心贡献是把三维几何监督从生成器侧转移到记忆侧，通过 camera-queryable geometry distillation 让固定大小的隐式 memory tokens 编码跨视角场景结构。实验显示，该设计在 MIND benchmark 上显著提升了长期视觉一致性和三维几何一致性，同时推理阶段保持轻量。

论文的方法设计清晰，实验围绕核心模块展开，结果有说服力。但其验证仍主要局限于单一 benchmark，硬件算力信息不充分，对真实世界复杂动态场景和更大规模长期记忆的泛化仍需进一步研究。
